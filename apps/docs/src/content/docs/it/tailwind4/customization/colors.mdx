# 색상

래퍼 요소의 CSS 사용자 정의 속성을 재정의하여 런타임에 색상 테마를 전환할 수 있습니다. Tailwind v4 패키지는 모든 색상을 시맨틱 토큰에서 읽어오므로, 기본 CSS 변수를 변경하면 전체 피커의 브랜드가 즉시 변경됩니다.

Tailwind v4 패키지는 시맨틱 토큰에 **oklch** 형식을 사용합니다. 피커의 브랜드를 변경하려면 `--color-primary` 및 `--color-primary-foreground`를 재정의하세요:
