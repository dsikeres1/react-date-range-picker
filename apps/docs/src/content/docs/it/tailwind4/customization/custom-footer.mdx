## Sovrascrittura delle Classi

Ogni parte composta accetta una prop `className`. Il pacchetto Tailwind v4 utilizza `cn()` (un'utilità `clsx` + `twMerge`) per unire le tue classi con quelle predefinite, in modo che le utilità in conflitto vengano risolte correttamente:

```tsx
import { DatePicker } from "react-date-range-picker-tailwind4";

<DatePicker.Root value={value} onChange={setValue}>
  {/* Sovrascrive la larghezza e il padding del trigger */}
  <DatePicker.Trigger className="min-w-[300px] px-4 py-3" />
  <DatePicker.Content className="rounded-2xl shadow-xl">
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title className="text-lg" />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>;
```
