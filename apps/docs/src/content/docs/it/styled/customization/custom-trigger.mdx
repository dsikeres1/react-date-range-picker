# 사용자 정의 트리거

기본 트리거를 원하는 버튼이나 입력 필드로 교체하세요. `Trigger` 컴포넌트는 렌더 프롭(render prop)을 통해 트리거의 모양과 동작을 완벽하게 제어할 수 있도록 지원합니다.

## 트리거 렌더 프롭

| Prop           | 타입             | 설명                      |
| -------------- | ---------------- | ------------------------- |
| `displayValue` | `string`         | 선택된 날짜의 포맷된 값   |
| `placeholder`  | `string`         | 플레이스홀더 텍스트       |
| `hasValue`     | `boolean`        | 날짜 선택 여부            |
| `isOpen`       | `boolean`        | 팝업 열림 여부            |
| `onToggle`     | `() => void`     | 팝업 열기/닫기 토글       |
| `onClear`      | `() => void`     | 선택 값 지우기            |
| `locale`       | `Locale`         | 현재 로케일 객체          |
| `triggerRef`   | `(node) => void` | 위치 지정을 위한 ref 콜백 |
