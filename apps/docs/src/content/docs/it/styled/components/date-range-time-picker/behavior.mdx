# 동작

## 선택 시 닫기

`shouldCloseOnSelect`가 활성화되면, 프리셋을 클릭했을 때 선택기가 자동으로 확인하고 닫힙니다.

<Admonition type="info">
  시간 설정이 활성화된 선택기에서 `shouldCloseOnSelect`는 프리셋 클릭에만 적용됩니다. 날짜 클릭에는
  적용되지 않는데, 이는 사용자가 아직 시간을 설정해야 하기 때문입니다.
</Admonition>
