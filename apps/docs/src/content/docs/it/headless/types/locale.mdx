# 로캘

모든 사용자 대면 문자열에 대한 인터페이스입니다. 특정 문자열을 재정의하려면 모든 훅에 `Partial<Locale>`을 전달하세요. 지정되지 않은 필드는 `DEFAULT_LOCALE`로 대체됩니다.

```tsx
import type { Locale } from "react-date-range-picker-headless";
```
