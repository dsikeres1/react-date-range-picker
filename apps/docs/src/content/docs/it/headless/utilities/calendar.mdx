# Utilità del Calendario

Funzioni per generare i dati della griglia del calendario.

```tsx
import {
  generateCalendarDays,
  daysToWeeks,
  buildCalendarMonth,
} from "react-date-range-picker-headless";
```

## generateCalendarDays

Genera un array piatto di 42 celle (6 settimane x 7 giorni) per un dato mese.
