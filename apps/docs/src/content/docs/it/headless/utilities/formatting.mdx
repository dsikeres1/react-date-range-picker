# 포매팅 유틸리티

날짜, 시간, 숫자를 표시에 맞게 포매팅하는 함수들입니다.

```tsx
import {
  formatDate,
  formatDateTime,
  formatTimeDisplay,
  padNumber,
} from "react-date-range-picker-headless";
```

## formatDate

사용자 정의 포맷 문자열로 날짜를 포매팅합니다.

|           |                                                   |
| --------- | ------------------------------------------------- |
| 시그니처  | `formatDate(date: Date, format?: string): string` |
| 기본 포맷 | `"YYYY.MM.DD"`                                    |

```tsx
formatDate(new Date(2026, 2, 1)); // "2026.03.01"
formatDate(new Date(2026, 2, 1), "YYYY-MM-DD"); // "2026-03-01"
formatDate(new Date(2026, 2, 1), "M/D"); // "3/1"
```

## formatDateTime

사용자 정의 포맷 문자열로 날짜와 시간을 포매팅합니다.

|           |                                                       |
| --------- | ----------------------------------------------------- |
| 시그니처  | `formatDateTime(date: Date, format?: string): string` |
| 기본 포맷 | `"YYYY.MM.DD HH:mm"`                                  |

```tsx
formatDateTime(new Date(2026, 2, 1, 14, 30)); // "2026.03.01 14:30"
formatDateTime(new Date(2026, 2, 1, 14, 30), "YYYY-MM-DD HH:mm:ss"); // "2026-03-01 14:30:00"
```

## formatTimeDisplay

결정된 `TimeConfig`에 기반하여 시간 표시 문자열을 포매팅합니다.
