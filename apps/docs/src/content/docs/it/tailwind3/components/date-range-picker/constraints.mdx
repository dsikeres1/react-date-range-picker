# Vincoli

## Giorni Minimi e Massimi

Limita la lunghezza dell'intervallo con `minDays` e `maxDays` (inclusi). Gli utenti non possono confermare un intervallo più corto di `minDays` o più lungo di `maxDays`.

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinMaxDays client:visible />
  </ExampleWrapper>
</CodeDemo>

## Date Disabilitate

Usa `isDateUnavailable` per disabilitare dinamicamente date specifiche. Le date disabilitate non possono essere selezionate come inizio o fine di un intervallo.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <DisabledDates client:visible />
  </ExampleWrapper>
</CodeDemo>

## Permetti Data Singola

Per impostazione predefinita, è consentito un intervallo in cui l'inizio è uguale alla fine. Imposta `allowSingleDateInRange={false}` per richiedere almeno due date distinte.

<CodeDemo source={AllowSingleDateSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <AllowSingleDate client:visible />
  </ExampleWrapper>
</CodeDemo>
