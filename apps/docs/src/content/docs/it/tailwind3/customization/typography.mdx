# 타이포그래피

Tailwind v3 패키지는 Tailwind의 기본 `font-sans` 클래스(`ui-sans-serif, system-ui, sans-serif, ...`)를 사용합니다. CSS `font-family`는 상속되므로, `fontFamily` 스타일이 적용된 컨테이너로 피커를 감싸 폰트를 재정의할 수 있습니다. 래퍼의 폰트는 모든 피커 요소에 계단식으로 적용됩니다.

## 글꼴 패밀리

피커를 감싸고 래퍼에 `fontFamily`를 설정하여 폰트를 변경하세요:

<CodeDemo source={CompoundCustomFontSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundCustomFont client:visible />
  </ExampleWrapper>
</CodeDemo>

## 웹 폰트 사용 (Noto Sans)

런타임에 웹 폰트를 로드하고 래퍼를 통해 적용합니다. 이 예제는 Google Fonts에서 [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans)를 로드합니다:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="프로덕션 사용법">
실제 프로젝트에서는 런타임에 주입하는 대신 HTML `<head>`의 `<link>` 태그나 CSS 번들러를 통해 폰트를 로드하세요. 여기서 사용된 `useEffect` 접근 방식은 데모용입니다.
</Admonition>
