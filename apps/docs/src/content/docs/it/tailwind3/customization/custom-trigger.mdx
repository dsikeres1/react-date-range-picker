# Trigger Personalizzato

Sostituisci il trigger predefinito con il tuo elemento personalizzato usando una render prop. Il componente `Trigger` accetta una funzione come figlio che riceve stato e callback, permettendoti di renderizzare qualsiasi pulsante o input personalizzato.

## Render Props del Trigger

| Prop           | Type             | Descrizione                        |
| -------------- | ---------------- | ---------------------------------- |
| `displayValue` | `string`         | Data selezionata formattata        |
| `placeholder`  | `string`         | Testo segnaposto                   |
| `hasValue`     | `boolean`        | Indica se una data ├© selezionata   |
| `isOpen`       | `boolean`        | Indica se il popup ├© aperto        |
| `onToggle`     | `() => void`     | Apre/chiude il popup               |
| `onClear`      | `() => void`     | Cancella la selezione              |
| `locale`       | `Locale`         | Oggetto locale corrente            |
| `triggerRef`   | `(node) => void` | Callback ref per il posizionamento |
