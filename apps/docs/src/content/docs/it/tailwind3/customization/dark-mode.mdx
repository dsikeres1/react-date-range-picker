# 다크 모드

Tailwind v3 패키지는 Tailwind의 `dark:` variant를 사용하여 완벽한 다크 모드 지원을 포함합니다. 모든 컴포넌트 요소에는 명시적인 다크 모드 클래스가 있습니다.

## 작동 방식

Tailwind v3 테마는 전반에 걸쳐 `dark:` variant를 사용하는 구체적인 색상 클래스를 사용합니다. 예를 들면 다음과 같습니다:

```
bg-white dark:bg-slate-950
text-slate-900 dark:text-slate-50
border-slate-200 dark:border-slate-800
```

다크 모드는 Tailwind의 `dark:` variant가 트리거될 때 활성화되며, 이는 Tailwind 설정의 `darkMode` 전략에 따라 달라집니다.

## Tailwind 설정

### 클래스 전략 (권장)

Tailwind 설정에서 `darkMode: "class"`로 설정하세요. 다크 모드는 `dark` 클래스가 `<html>` 또는 부모 요소에 있을 때 활성화됩니다:

```js
// tailwind.config.js
module.exports = {
  darkMode: "class",
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
};
```

다크 모드 토글:

```tsx
function App() {
  const [dark, setDark] = useState(false);

  return (
    <div className={dark ? "dark" : ""}>
      <button onClick={() => setDark(!dark)}>Toggle Theme</button>
      <DatePicker value={value} onChange={setValue} />
    </div>
  );
}
```

또는 `<html>`에 클래스를 적용합니다:

```tsx
document.documentElement.classList.toggle("dark", isDark);
```

### Selector 전략

Tailwind v3.4+는 `darkMode: "selector"`를 지원하여 `[data-theme="dark"]` 또는 다른 커스텀 selector를 사용할 수 있게 해줍니다:

```js
// tailwind.config.js
module.exports = {
  darkMode: ["selector", '[data-theme="dark"]'],
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
};
```

```tsx
<div data-theme={dark ? "dark" : "light"}>
  <DatePicker value={value} onChange={setValue} />
</div>
```

### Media 전략

`darkMode: "media"`로 설정하거나 생략하여(기본값) OS 설정을 따르게 할 수 있습니다:

```js
// tailwind.config.js
module.exports = {
  darkMode: "media",
  // ...
};
```

추가 설정이 필요하지 않습니다. 피커는 `prefers-color-scheme`를 따릅니다.

## 색상 쌍

피커 전반에 사용되는 라이트/다크 색상 쌍은 다음과 같습니다:

| 요소        | 라이트              | 다크                         |
| ----------- | ------------------- | ---------------------------- |
| 배경        | `bg-white`          | `dark:bg-slate-950`          |
| 텍스트      | `text-slate-900`    | `dark:text-slate-50`         |
| 흐린 텍스트 | `text-slate-500`    | `dark:text-slate-400`        |
| 테두리      | `border-slate-200`  | `dark:border-slate-800`      |
| Hover 배경  | `bg-slate-100`      | `dark:bg-slate-800`          |
| 주요 색상   | `bg-sky-500`        | `dark:bg-sky-500`            |
| 주요 텍스트 | `text-sky-500`      | `dark:text-sky-400`          |
| 범위 배경   | `bg-sky-50`         | `dark:bg-sky-950/50`         |
| Hover 범위  | `bg-sky-50/50`      | `dark:bg-sky-950/30`         |
| 포커스 링   | `ring-sky-500`      | `ring-sky-500`               |
| 링 오프셋   | `ring-offset-white` | `dark:ring-offset-slate-950` |

## Docusaurus와 통합

Docusaurus(`<html>`에 `data-theme`를 사용하는)를 사용 중이라면 Tailwind의 selector 전략을 설정하세요:

```js
// tailwind.config.js
module.exports = {
  darkMode: ["selector", '[data-theme="dark"]'],
  // ...
};
```

Docusaurus는 다크 모드가 활성화되면 `<html data-theme="dark">`를 설정하며, Tailwind의 `dark:` variant는 이 selector에서 트리거됩니다.

## next-themes와 통합

```tsx
// app/layout.tsx
import { ThemeProvider } from "next-themes";

export default function RootLayout({ children }) {
  return (
    <html suppressHydrationWarning>
      <body>
        <ThemeProvider attribute="class" defaultTheme="system">
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

`next-themes`는 `<html>`에서 `dark` 클래스를 추가/제거합니다. Tailwind 설정에 `darkMode: "class"`가 있으면 피커가 자동으로 전환됩니다.

## 특정 모드 강제하기

피커가 항상 라이트 또는 다크 모드를 사용하도록 강제하려면 적절한 클래스를 가진 컨테이너로 감싸주세요:

```tsx
{
  /* 항상 라이트 — "dark" 클래스 조상 없음 */
}
<div className="">
  <DatePicker value={value} onChange={setValue} />
</div>;

{
  /* 항상 다크 */
}
<div className="dark">
  <DatePicker value={value} onChange={setValue} />
</div>;
```

<Admonition type="note">
`darkMode: "class"`를 사용할 때, `dark` 클래스는 부모 요소나 `<html>`에 있어야 합니다. 피커 자체는 다크 모드 토글을 관리하지 않습니다.
</Admonition>
