# 색상

Tailwind v3 패키지는 CSS 변수 대신 **하드코딩된 Tailwind 유틸리티 클래스**(예: `bg-sky-500`)를 사용합니다. 색상을 바꾸려면 Compound API를 사용하여 동적으로 요일 클래스 이름을 재정의해야 합니다:

<Admonition type="info">
  `styled` 패키지(CSS 변수 사용)와 달리, Tailwind v3는 런타임에 색상을 재정의하기 위해 Compound
  API가 필요합니다. 자세한 내용은 [테마 재정의](/docs/tailwind3/customization/theme-override)를
  참조하세요.
</Admonition>
