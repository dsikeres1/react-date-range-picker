# Utilidades de Formateo

Funciones para formatear fechas, horas y números para su visualización.

```tsx
import {
  formatDate,
  formatDateTime,
  formatTimeDisplay,
  padNumber,
} from "react-date-range-picker-headless";
```

## formatDate

Formatea una fecha con una cadena de formato personalizable.

|                     |                                                   |
| ------------------- | ------------------------------------------------- |
| Firma               | `formatDate(date: Date, format?: string): string` |
| Formato por defecto | `"YYYY.MM.DD"`                                    |

```tsx
formatDate(new Date(2026, 2, 1)); // "2026.03.01"
formatDate(new Date(2026, 2, 1), "YYYY-MM-DD"); // "2026-03-01"
formatDate(new Date(2026, 2, 1), "M/D"); // "3/1"
```

## formatDateTime

Formatea una fecha con hora usando una cadena de formato personalizable.

|                     |                                                       |
| ------------------- | ----------------------------------------------------- |
| Firma               | `formatDateTime(date: Date, format?: string): string` |
| Formato por defecto | `"YYYY.MM.DD HH:mm"`                                  |

```tsx
formatDateTime(new Date(2026, 2, 1, 14, 30)); // "2026.03.01 14:30"
formatDateTime(new Date(2026, 2, 1, 14, 30), "YYYY-MM-DD HH:mm:ss"); // "2026-03-01 14:30:00"
```
