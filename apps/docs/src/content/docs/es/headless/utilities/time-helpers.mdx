# Ayudantes de Tiempo

Funciones para generar valores de tiempo, convertir entre formatos de 12/24 horas, ajustar a pasos y resolver la configuración de tiempo.

```tsx
import {
  generateHours,
  generateHours12,
  generateMinutes,
  generateSeconds,
  adjustMinuteToStep,
  adjustSecondToStep,
  to12Hour,
  to24Hour,
  resolveTimeConfig,
} from "react-date-range-picker-headless";
```

## generateHours

Genera un array de valores en formato de 24 horas.

|          |                             |
| -------- | --------------------------- |
| Firma    | `generateHours(): number[]` |
| Devuelve | `[0, 1, 2, ..., 23]`        |

## generateHours12

Genera un array de valores en formato de 12 horas.

|          |                               |
| -------- | ----------------------------- |
| Firma    | `generateHours12(): number[]` |
| Devuelve | `[1, 2, 3, ..., 12]`          |

## generateMinutes

Genera un array de minutos filtrado por el paso (step).

|             |                                                      |
| ----------- | ---------------------------------------------------- |
| Firma       | `generateMinutes(minuteStep?: MinuteStep): number[]` |
| Por defecto | `5`                                                  |

```tsx
generateMinutes(); // [0, 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55]
generateMinutes(15); // [0, 15, 30, 45]
generateMinutes(1); // [0, 1, 2, ..., 59]
```

## generateSeconds

Genera un array de segundos filtrado por el paso (step).

|             |                                                      |
| ----------- | ---------------------------------------------------- |
| Firma       | `generateSeconds(secondStep?: SecondStep): number[]` |
| Por defecto | `1`                                                  |

```tsx
generateSeconds(); // [0, 1, 2, ..., 59]
generateSeconds(15); // [0, 15, 30, 45]
```

## adjustMinuteToStep

Ajusta un valor de minuto al paso válido más cercano (redondea hacia abajo).

|       |                                                                      |
| ----- | -------------------------------------------------------------------- |
| Firma | `adjustMinuteToStep(minute: number, minuteStep: MinuteStep): number` |

```tsx
adjustMinuteToStep(7, 5); // 5
adjustMinuteToStep(13, 10); // 10
adjustMinuteToStep(29, 15); // 15
```

## adjustSecondToStep

Ajusta un valor de segundo al paso válido más cercano (redondea hacia abajo).

|       |                                                                      |
| ----- | -------------------------------------------------------------------- |
| Firma | `adjustSecondToStep(second: number, secondStep: SecondStep): number` |

```tsx
adjustSecondToStep(7, 5); // 5
adjustSecondToStep(14, 10); // 10
```

## to12Hour

Convierte un valor en formato de 24 horas a formato de 12 horas con AM/PM.

|       |                                                                  |
| ----- | ---------------------------------------------------------------- |
| Firma | `to12Hour(hour24: number): { hour: number; period: TimePeriod }` |

```tsx
to12Hour(0); // { hour: 12, period: "AM" }
to12Hour(12); // { hour: 12, period: "PM" }
to12Hour(13); // { hour: 1, period: "PM" }
to12Hour(23); // { hour: 11, period: "PM" }
```

## to24Hour

Convierte un valor en formato de 12 horas con AM/PM a formato de 24 horas.

|       |                                                        |
| ----- | ------------------------------------------------------ |
| Firma | `to24Hour(hour12: number, period: TimePeriod): number` |

```tsx
to24Hour(12, "AM"); // 0
to24Hour(12, "PM"); // 12
to24Hour(1, "PM"); // 13
to24Hour(11, "PM"); // 23
```

## resolveTimeConfig

Resuelve un `TimeConfig` parcial con valores por defecto.

|       |                                                                |
| ----- | -------------------------------------------------------------- |
| Firma | `resolveTimeConfig(config?: TimeConfig): Required<TimeConfig>` |

| Campo        | Por defecto |
| ------------ | ----------- |
| `precision`  | `"minute"`  |
| `hourFormat` | `"24"`      |
| `minuteStep` | `5`         |
| `secondStep` | `1`         |
| `itemHeight` | `32`        |

```tsx
resolveTimeConfig();
// { precision: "minute", hourFormat: "24", minuteStep: 5, secondStep: 1, itemHeight: 32 }

resolveTimeConfig({ precision: "second", hourFormat: "12" });
// { precision: "second", hourFormat: "12", minuteStep: 5, secondStep: 1, itemHeight: 32 }
```
