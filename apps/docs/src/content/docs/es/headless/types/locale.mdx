# Localización

Interfaz para todas las cadenas de texto orientadas al usuario. Pasa un `Partial<Locale>` a cualquier hook para anular cadenas específicas -- los campos no especificados recurren a `DEFAULT_LOCALE`.

```tsx
import type { Locale } from "react-date-range-picker-headless";
```

## Campos

| Campo      | Tipo       | Descripción |
| ---------- | ---------- | ----------- |
| `weekdays` | `string[]` |
