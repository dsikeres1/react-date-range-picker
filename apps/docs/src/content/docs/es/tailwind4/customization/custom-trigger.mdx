# Trigger Personalizado

Reemplaza el botón de disparo (trigger) por defecto con tu propio elemento usando el render prop `Trigger`. Cuando pasas una función como hijo a `DatePicker.Trigger`, recibes el estado y callbacks para construir cualquier UI de trigger que desees.

    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Render Props del Trigger

| Prop           | Tipo             | Descripción                            |
| -------------- | ---------------- | -------------------------------------- |
| `displayValue` | `string`         | Fecha seleccionada formateada          |
| `placeholder`  | `string`         | Texto del placeholder                  |
| `hasValue`     | `boolean`        | Si se ha seleccionado una fecha        |
| `isOpen`       | `boolean`        | Si el popup está abierto               |
| `onToggle`     | `() => void`     | Alterna la apertura/cierre del popup   |
| `onClear`      | `() => void`     | Limpia la selección                    |
| `locale`       | `Locale`         | Objeto de localización (locale) actual |
| `triggerRef`   | `(node) => void` | Callback de ref para posicionamiento   |
