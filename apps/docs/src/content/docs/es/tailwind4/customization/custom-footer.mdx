## Sobrescribir Clases

Cada parte compuesta acepta una prop `className`. El paquete de Tailwind v4 utiliza `cn()` (una utilidad de `clsx` + `twMerge`) para fusionar tus clases con las predeterminadas, de modo que las utilidades en conflicto se resuelven correctamente:

```tsx
import { DatePicker } from "react-date-range-picker-tailwind4";

<DatePicker.Root value={value} onChange={setValue}>
  {/* Override trigger width and padding */}
  <DatePicker.Trigger className="min-w-[300px] px-4 py-3" />
  <DatePicker.Content className="rounded-2xl shadow-xl">
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title className="text-lg" />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>;
```
