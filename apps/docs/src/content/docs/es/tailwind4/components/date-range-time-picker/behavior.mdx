# Comportamiento

## Cerrar al Seleccionar

Cuando `shouldCloseOnSelect` está activado, el selector se confirma y cierra automáticamente al hacer clic en un preajuste.

<Admonition type="info">
  Para los selectores con hora activada, `shouldCloseOnSelect` solo se aplica a los clics en
  preajustes, no a los clics en fechas, ya que el usuario aún necesita establecer la hora.
</Admonition>
