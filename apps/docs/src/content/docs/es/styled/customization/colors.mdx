# Colores

Cambia el tema de color en tiempo de ejecución sobrescribiendo las propiedades personalizadas de CSS en un elemento contenedor. El paquete styled utiliza `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light` y `--rdrp-color-primary-lighter` para todos los colores de acento primarios.
