# Trigger Personalizado

Reemplaza el disparador predeterminado con tu propio botón o entrada. El componente `Trigger` acepta una render prop que te da control total sobre la apariencia y el comportamiento del disparador.

  <ExampleWrapper variant="styled" client:load>
    <CompoundCustomTrigger client:load />
  </ExampleWrapper>
</CodeDemo>

## Render Props del Trigger

| Prop           | Tipo             | Descripción                          |
| -------------- | ---------------- | ------------------------------------ |
| `displayValue` | `string`         | Fecha seleccionada formateada        |
| `placeholder`  | `string`         | Texto del placeholder                |
| `hasValue`     | `boolean`        | Si se ha seleccionado una fecha      |
| `isOpen`       | `boolean`        | Si el popup está abierto             |
| `onToggle`     | `() => void`     | Alterna abrir/cerrar el popup        |
| `onClear`      | `() => void`     | Limpia la selección                  |
| `locale`       | `Locale`         | Objeto de localización actual        |
| `triggerRef`   | `(node) => void` | Callback de ref para posicionamiento |
