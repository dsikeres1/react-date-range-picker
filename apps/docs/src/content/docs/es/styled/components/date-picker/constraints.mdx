# Restricciones

## Requerido

Cuando se establece `required`, el botón de limpiar se deshabilita y el valor no se puede vaciar.

<CodeDemo source={RequiredSource}>
  <ExampleWrapper variant="styled" client:visible>
    <Required client:visible />
  </ExampleWrapper>
</CodeDemo>

## Fechas Mínima y Máxima

Restringe las fechas seleccionables a un rango específico usando `minDate` y `maxDate`.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="styled" client:visible>
    <MinMaxDate client:visible />
  </ExampleWrapper>
</CodeDemo>

## Fechas Deshabilitadas

Usa `isDateUnavailable` para deshabilitar dinámicamente fechas específicas.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="styled" client:visible>
    <DisabledDates client:visible />
  </ExampleWrapper>
</CodeDemo>

## Deshabilitar Pasado / Futuro

Usa `disablePast` o `disableFuture` para deshabilitar rápidamente fechas antes/después de hoy.

  <ExampleWrapper variant="styled" client:visible>
    <DisablePastFuture client:visible />
