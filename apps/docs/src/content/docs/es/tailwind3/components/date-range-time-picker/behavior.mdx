# Comportamiento

## Cerrar al seleccionar

Cuando `shouldCloseOnSelect` está habilitado, el selector se confirma y cierra automáticamente cuando se hace clic en un preajuste.

<Admonition type="info">
  Para selectores con hora habilitada, `shouldCloseOnSelect` solo se aplica a los clics en
  preajustes, no a los clics en fechas, ya que el usuario aún necesita establecer la hora.
</Admonition>
