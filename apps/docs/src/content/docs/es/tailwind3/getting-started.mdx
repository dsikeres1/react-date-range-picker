# Primeros Pasos con Tailwind CSS v3

El paquete de Tailwind v3 proporciona componentes de selector de fechas estilizados con clases de utilidad de Tailwind CSS v3.

## Instalación

<PackageInstall pkg="react-date-range-picker-tailwind3" />

Importa los estilos del componente en el punto de entrada de tu aplicación:

```tsx
import "react-date-range-picker-tailwind3/rdrp-styles.css";
```

## Configuración de Tailwind

Añade el paquete a tus rutas de `content` en `tailwind.config.js` para que Tailwind pueda escanear las clases de los componentes:

```js
// tailwind.config.js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
  // ...
};
```
