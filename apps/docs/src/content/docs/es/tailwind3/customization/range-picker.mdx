# Selector de Rango

La API compuesta `DateRangePicker` proporciona partes adicionales para calendarios duales y rangos preestablecidos. Obtienes control total sobre el diseño de los preajustes, las cuadrículas del calendario, los encabezados y las acciones del pie de página.
