# Colores

El paquete de Tailwind v3 utiliza **clases de utilidad de Tailwind codificadas** (p. ej., `bg-sky-500`) en lugar de variables CSS. Para cambiar los colores, utiliza la API Compound y sobrescribe los nombres de las clases de los días dinámicamente:

<Admonition type="info">
  A diferencia del paquete styled (que utiliza variables CSS), Tailwind v3 requiere la API Compound
  para sobrescribir los colores en tiempo de ejecución. Consulta [Sobrescritura de
  Tema](/docs/tailwind3/customization/theme-override) para más detalles.
</Admonition>
