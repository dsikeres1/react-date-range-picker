# Disparador Personalizado

Reemplaza el disparador predeterminado con tu propio elemento usando una render prop. El componente `Trigger` acepta un hijo de función que recibe estado y callbacks, permitiéndote renderizar cualquier botón o entrada personalizada.

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <CompoundCustomTrigger client:load />
  </ExampleWrapper>
</CodeDemo>

## Render Props del Disparador

| Prop           | Tipo             | Descripción                            |
| -------------- | ---------------- | -------------------------------------- |
| `displayValue` | `string`         | Fecha seleccionada formateada          |
| `placeholder`  | `string`         | Texto de marcador de posición          |
| `hasValue`     | `boolean`        | Indica si se ha seleccionado una fecha |
| `isOpen`       | `boolean`        | Indica si el popup está abierto        |
| `onToggle`     | `() => void`     | Abre/cierra el popup                   |
| `onClear`      | `() => void`     | Limpia la selección                    |
| `locale`       | `Locale`         | Objeto de localización actual          |
| `triggerRef`   | `(node) => void` | Callback de ref para posicionamiento   |
