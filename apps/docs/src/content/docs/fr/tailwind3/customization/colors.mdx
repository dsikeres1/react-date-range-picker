# Couleurs

Le paquet Tailwind v3 utilise des **classes utilitaires Tailwind codées en dur** (par ex., `bg-sky-500`) au lieu de variables CSS. Pour changer les couleurs, utilisez l'API Compound et surchargez dynamiquement les noms de classe des jours :

<Admonition type="info">
  Contrairement au paquet styled (qui utilise des variables CSS), Tailwind v3 nécessite l'API
  Compound pour surcharger les couleurs à l'exécution. Voir [Surcharge de
  Thème](/docs/tailwind3/customization/theme-override) pour plus de détails.
</Admonition>
