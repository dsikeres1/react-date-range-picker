# Comportement

## Fermeture à la Sélection

Lorsque `shouldCloseOnSelect` est activé, le sélecteur confirme et se ferme automatiquement lorsqu'un préréglage est cliqué.

<Admonition type="info">
  Pour les sélecteurs avec heure, `shouldCloseOnSelect` s'applique uniquement aux clics sur les
  préréglages — pas aux clics sur les dates, car l'utilisateur doit encore définir l'heure.
</Admonition>
