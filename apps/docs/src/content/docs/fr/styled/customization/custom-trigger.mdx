# Déclencheur Personnalisé

Remplacez le déclencheur par défaut par votre propre bouton ou champ de saisie. Le composant `Trigger` accepte une render prop qui vous donne un contrôle total sur l'apparence et le comportement du déclencheur.

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Props de Rendu du Déclencheur

| Propriété      | Type             | Description                          |
| -------------- | ---------------- | ------------------------------------ |
| `displayValue` | `string`         | Date sélectionnée formatée           |
| `placeholder`  | `string`         | Texte indicatif                      |
| `hasValue`     | `boolean`        | Indique si une date est sélectionnée |
| `isOpen`       | `boolean`        | Indique si la popup est ouverte      |
| `onToggle`     | `() => void`     | Ouvre/ferme la popup                 |
| `onClear`      | `() => void`     | Efface la sélection                  |
| `locale`       | `Locale`         | Objet de la locale actuelle          |
| `triggerRef`   | `(node) => void` | Ref de rappel pour le positionnement |
