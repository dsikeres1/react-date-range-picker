# Cellule de Jour Personnalisée

Utilisez la render prop du composant `Day` pour personnaliser l'apparence de chaque cellule de jour. C'est utile pour ajouter des indicateurs d'événements, des infobulles, ou un style complètement personnalisé à des jours individuels.
