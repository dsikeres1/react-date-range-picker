# Couleurs

Changez le thème de couleur à l'exécution en surchargeant les propriétés personnalisées CSS sur un élément englobant. Le paquet styled utilise `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light`, et `--rdrp-color-primary-lighter` pour toutes les couleurs d'accentuation principales.

## Sélecteur de Thème Intégré

  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

## Sélecteur de Thème Popup

La même approche fonctionne avec les sélecteurs popup :

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="styled" client:visible>

  </ExampleWrapper>

## Style en Ligne avec Variable CSS

Vous pouvez également surcharger les couleurs directement via l'attribut `style` sur un élément englobant :

  <ExampleWrapper variant="styled" client:visible>
    <CSSCustomColors client:visible />
  </ExampleWrapper>
</CodeDemo>
