# Couleurs

Changez les thèmes de couleur à l'exécution en surchargeant les propriétés personnalisées CSS sur un élément conteneur. Étant donné que le package Tailwind v4 lit toutes les couleurs à partir de jetons sémantiques, la modification des variables CSS sous-jacentes change instantanément l'apparence de tout le sélecteur.

Le package Tailwind v4 utilise le format **oklch** pour les jetons sémantiques. Surchargez `--color-primary` et `--color-primary-foreground` pour changer l'apparence du sélecteur :

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <CompoundColorTheme client:load />
  </ExampleWrapper>
</CodeDemo>
