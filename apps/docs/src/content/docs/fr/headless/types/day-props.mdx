# DayProps

Drapeaux calculés par cellule retournés par `getDayProps()`. Décrit l'état visuel et interactif d'une seule cellule de jour du calendrier.

```tsx
import type { DayProps } from "react-date-range-picker-headless";
```

## Champs

| Champ  | Type   | Description                      |
| ------ | ------ | -------------------------------- |
| `date` | `Date` | L'objet Date pour cette cellule. |

|
