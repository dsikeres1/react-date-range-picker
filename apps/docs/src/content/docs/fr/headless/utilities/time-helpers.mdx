# Assistants Temporels

Fonctions pour générer des valeurs de temps, convertir entre les formats 12/24 heures, s'aligner sur des pas, et résoudre la configuration du temps.

```tsx
import {
  generateHours,
  generateHours12,
  generateMinutes,
  generateSeconds,
  adjustMinuteToStep,
  adjustSecondToStep,
  to12Hour,
  to24Hour,
  resolveTimeConfig,
} from "react-date-range-picker-headless";
```

## generateHours

Générer un tableau de valeurs sur 24 heures.

|           |                             |
| --------- | --------------------------- |
| Signature | `generateHours(): number[]` |
| Retourne  | `[0, 1, 2, ..., 23]`        |

## generateHours12

Générer un tableau de valeurs sur 12 heures.

|           |                               |
| --------- | ----------------------------- |
| Signature | `generateHours12(): number[]` |
| Retourne  | `[1, 2, 3, ..., 12]`          |

## generateMinutes

Générer un tableau de minutes filtré par pas.

|           |                                                      |
| --------- | ---------------------------------------------------- |
| Signature | `generateMinutes(minuteStep?: MinuteStep): number[]` |
| Défaut    | `5`                                                  |

```tsx
generateMinutes(); // [0, 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55]
generateMinutes(15); // [0, 15, 30, 45]
generateMinutes(1); // [0, 1, 2, ..., 59]
```

## generateSeconds

Générer un tableau de secondes filtré par pas.

|           |                                                      |
| --------- | ---------------------------------------------------- |
| Signature | `generateSeconds(secondStep?: SecondStep): number[]` |
| Défaut    | `1`                                                  |

```tsx
generateSeconds(); // [0, 1, 2, ..., 59]
generateSeconds(15); // [0, 15, 30, 45]
```

## adjustMinuteToStep

Arrondir une valeur de minute au pas valide le plus proche (arrondi à l'inférieur).

|           |                                                                      |
| --------- | -------------------------------------------------------------------- |
| Signature | `adjustMinuteToStep(minute: number, minuteStep: MinuteStep): number` |

```tsx
adjustMinuteToStep(7, 5); // 5
adjustMinuteToStep(13, 10); // 10
adjustMinuteToStep(29, 15); // 15
```

## adjustSecondToStep

Arrondir une valeur de seconde au pas valide le plus proche (arrondi à l'inférieur).

|           |                                                                      |
| --------- | -------------------------------------------------------------------- |
| Signature | `adjustSecondToStep(second: number, secondStep: SecondStep): number` |

```tsx
adjustSecondToStep(7, 5); // 5
adjustSecondToStep(14, 10); // 10
```

## to12Hour

Convertir une valeur sur 24 heures au format 12 heures avec AM/PM.

|           |                                                                  |
| --------- | ---------------------------------------------------------------- |
| Signature | `to12Hour(hour24: number): { hour: number; period: TimePeriod }` |

```tsx
to12Hour(0); // { hour: 12, period: "AM" }
to12Hour(12); // { hour: 12, period: "PM" }
to12Hour(13); // { hour: 1, period: "PM" }
to12Hour(23); // { hour: 11, period: "PM" }
```

## to24Hour

Convertir une valeur sur 12 heures avec AM/PM au format 24 heures.

|           |                                                        |
| --------- | ------------------------------------------------------ |
| Signature | `to24Hour(hour12: number, period: TimePeriod): number` |

```tsx
to24Hour(12, "AM"); // 0
to24Hour(12, "PM"); // 12
to24Hour(1, "PM"); // 13
to24Hour(11, "PM"); // 23
```

## resolveTimeConfig

Résoudre une `TimeConfig` partielle avec les valeurs par défaut.

|           |                                                                |
| --------- | -------------------------------------------------------------- |
| Signature | `resolveTimeConfig(config?: TimeConfig): Required<TimeConfig>` |

| Champ        | Défaut     |
| ------------ | ---------- |
| `precision`  | `"minute"` |
| `hourFormat` | `"24"`     |
| `minuteStep` | `5`        |
| `secondStep` | `1`        |
| `itemHeight` | `32`       |

```tsx
resolveTimeConfig();
// { precision: "minute", hourFormat: "24", minuteStep: 5, secondStep: 1, itemHeight: 32 }

resolveTimeConfig({ precision: "second", hourFormat: "12" });
// { precision: "second", hourFormat: "12", minuteStep: 5, secondStep: 1, itemHeight: 32 }
```
