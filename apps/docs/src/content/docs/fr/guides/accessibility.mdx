# Accessibilité

Tous les composants de sélection sont livrés avec un support d'accessibilité intégré. La navigation au clavier, les attributs ARIA et la gestion du focus fonctionnent d'emblée dans chaque variante — Styled, Tailwind v3/v4 et Headless.

Aucune configuration supplémentaire n'est nécessaire pour une utilisation standard. Cette page documente ce qui est inclus afin que vous puissiez auditer, étendre ou répliquer les modèles dans des interfaces utilisateur personnalisées.
