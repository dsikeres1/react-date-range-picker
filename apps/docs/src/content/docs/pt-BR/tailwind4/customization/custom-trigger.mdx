# Gatilho Personalizado

Substitua o botão de gatilho padrão pelo seu próprio elemento usando a render prop `Trigger`. Ao passar uma função como filho para o `DatePicker.Trigger`, você recebe estado e callbacks para construir qualquer interface de gatilho que desejar.

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Render Props do Gatilho

| Prop           | Tipo             | Descrição                              |
| -------------- | ---------------- | -------------------------------------- |
| `displayValue` | `string`         | Data selecionada formatada             |
| `placeholder`  | `string`         | Texto do placeholder                   |
| `hasValue`     | `boolean`        | Se uma data está selecionada           |
| `isOpen`       | `boolean`        | Se o popup está aberto                 |
| `onToggle`     | `() => void`     | Alterna a abertura/fechamento do popup |
| `onClear`      | `() => void`     | Limpa a seleção                        |
| `locale`       | `Locale`         | Objeto de localidade atual             |
| `triggerRef`   | `(node) => void` | Callback de ref para posicionamento    |
