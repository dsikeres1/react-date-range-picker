# Cores

Troque os temas de cores em tempo de execução sobrescrevendo as propriedades personalizadas de CSS em um elemento wrapper. Como o pacote Tailwind v4 lê todas as cores de tokens semânticos, alterar as variáveis CSS subjacentes reformula instantaneamente todo o seletor.

O pacote Tailwind v4 usa o formato **oklch** para tokens semânticos. Sobrescreva `--color-primary` e `--color-primary-foreground` para reformular o seletor:

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

## Modo Popup

A mesma técnica funciona com seletores popup:

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorThemePopup client:visible />
  </ExampleWrapper>
</CodeDemo>
