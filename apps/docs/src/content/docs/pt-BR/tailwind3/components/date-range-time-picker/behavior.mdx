# Comportamento

## Fechar ao Selecionar

Quando `shouldCloseOnSelect` está ativado, o seletor confirma e fecha automaticamente quando uma predefinição é clicada.

<Admonition type="info">
  Para seletores com tempo ativado, `shouldCloseOnSelect` se aplica apenas a cliques em
  predefinições — não a cliques em datas, pois o usuário ainda precisa definir a hora.
</Admonition>
