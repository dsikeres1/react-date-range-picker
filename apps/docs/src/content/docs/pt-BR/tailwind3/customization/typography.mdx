# Tipografia

O pacote Tailwind v3 usa a classe padrão `font-sans` do Tailwind (`ui-sans-serif, system-ui, sans-serif, ...`). Como o `font-family` do CSS é herdado, você pode substituir a fonte do seletor envolvendo-o em um contêiner com um estilo `fontFamily`. A fonte do contêiner é aplicada em cascata a todos os elementos do seletor.

## Família da Fonte

Envolva o seletor e defina `fontFamily` no contêiner para trocar de fonte:

## Usando Web Fonts (Noto Sans)

Carregue uma web font em tempo de execução e aplique-a através de um contêiner. Este exemplo carrega a [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) do Google Fonts:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="Uso em Produção">
Em um projeto real, carregue a fonte através de uma tag `<link>` no `<head>` do seu HTML ou através do seu empacotador de CSS, em vez de injetá-la em tempo de execução. A abordagem com `useEffect` aqui é apenas para fins de demonstração.
</Admonition>
