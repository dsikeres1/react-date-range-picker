# Cores

O pacote Tailwind v3 usa **classes utilitárias do Tailwind codificadas** (ex: `bg-sky-500`) em vez de variáveis CSS. Para trocar as cores, use a API Compound e sobrescreva os nomes das classes dos dias dinamicamente:

<Admonition type="info">
  Diferente do pacote styled (que usa variáveis CSS), o Tailwind v3 requer a API Compound para
  sobrescrever cores em tempo de execução. Veja [Sobrescrita de
  Tema](/docs/tailwind3/customization/theme-override) para mais detalhes.
</Admonition>
