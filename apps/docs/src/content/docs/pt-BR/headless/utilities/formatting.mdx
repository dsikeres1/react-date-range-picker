# Utilitários de Formatação

Funções para formatar datas, horas e números para exibição.

```tsx
import {
  formatDate,
  formatDateTime,
  formatTimeDisplay,
  padNumber,
} from "react-date-range-picker-headless";
```

## formatDate

Formata uma data com uma string de formato personalizável.

|                |                                                   |
| -------------- | ------------------------------------------------- |
| Assinatura     | `formatDate(date: Date, format?: string): string` |
| Formato padrão | `"YYYY.MM.DD"`                                    |

```tsx
formatDate(new Date(2026, 2, 1)); // "2026.03.01"
formatDate(new Date(2026, 2, 1), "YYYY-MM-DD"); // "2026-03-01"
formatDate(new Date(2026, 2, 1), "M/D"); // "3/1"
```

## formatDateTime

Formata uma data com hora usando uma string de formato personalizável.

|                |                                                       |
| -------------- | ----------------------------------------------------- |
| Assinatura     | `formatDateTime(date: Date, format?: string): string` |
| Formato padrão | `"YYYY.MM.DD HH:mm"`                                  |

```tsx
formatDateTime(new Date(2026, 2, 1, 14, 30)); // "2026.03.01 14:30"
formatDateTime(new Date(2026, 2, 1, 14, 30), "YYYY-MM-DD HH:mm:ss"); // "2026-03-01 14:30:00"
```
