# DayProps

Flags computados por célula retornados por `getDayProps()`. Descreve o estado visual e interativo de uma única célula de dia do calendário.

```tsx
import type { DayProps } from "react-date-range-picker-headless";
```

## Campos

| Campo  | Tipo   | Descrição                       |
| ------ | ------ | ------------------------------- |
| `date` | `Date` | O objeto Date para esta célula. |
| `day`  |
