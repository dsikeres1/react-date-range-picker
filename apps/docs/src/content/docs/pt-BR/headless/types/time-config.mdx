# TimeConfig

Objeto de configuração para seletores relacionados a tempo. Compartilhado entre `useDateTimePicker`, `useDateRangeTimePicker` e `useStandaloneTimePicker`.

```tsx
import type { TimeConfig } from "react-date-range-picker-headless";
```

## Campos

| Campo        | Tipo            | Padrão     | Descrição                                                          |
| ------------ | --------------- | ---------- | ------------------------------------------------------------------ |
| `precision`  | `TimePrecision` | `"minute"` | Precisão de exibição da hora: `"hour"`, `"minute"`, ou `"second"`. |
| `hourFormat` | `HourFormat`    | `"24"`     | Formato de 12 horas ou 24 horas: `"12"` ou `"24"`.                 |
| `minuteStep` | `MinuteStep`    | `5`        | Passo de incremento dos minutos.                                   |
| `secondStep` | `SecondStep`    | `1`        | Passo de incremento dos segundos.                                  |
| `itemHeight` | `number`        | `32`       | Altura (em px) de cada item da roda de rolagem.                    |

## Definição TypeScript

```tsx
interface TimeConfig {
  precision?: TimePrecision;
  hourFormat?: HourFormat;
  minuteStep?: MinuteStep;
  secondStep?: SecondStep;
  itemHeight?: number;
}
```

## Tipos Relacionados

```tsx
type TimePrecision = "hour" | "minute" | "second";
type HourFormat = "12" | "24";
type MinuteStep = 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
type SecondStep = 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
```

## Configuração Resolvida

Hooks que aceitam `TimeConfig` retornam uma `resolvedTimeConfig` com todos os padrões preenchidos:

```tsx
type Required<TimeConfig>
// { precision: TimePrecision; hourFormat: HourFormat; minuteStep: MinuteStep; secondStep: SecondStep; itemHeight: number }
```

Use o utilitário `resolveTimeConfig()` para resolver os padrões manualmente:

```tsx
import { resolveTimeConfig } from "react-date-range-picker-headless";

const resolved = resolveTimeConfig({ precision: "second" });
// { precision: "second", hourFormat: "24", minuteStep: 5, secondStep: 1, itemHeight: 32 }
```
