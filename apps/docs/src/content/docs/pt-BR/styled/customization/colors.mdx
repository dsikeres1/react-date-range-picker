# Cores

Altere o tema de cores em tempo de execução sobrescrevendo as propriedades personalizadas (custom properties) de CSS em um elemento wrapper. O pacote styled usa `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light`, e `--rdrp-color-primary-lighter` para todas as cores de destaque primárias.

## Seletor de Tema Inline

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

## Seletor de Tema em Popup

A mesma abordagem funciona com seletores em popup:

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorThemePopup client:visible />
  </ExampleWrapper>
</CodeDemo>

## Estilo Inline com Variáveis CSS

Você também pode sobrescrever as cores diretamente através do atributo `style` em um elemento wrapper:

<CodeDemo source={CSSCustomColorsSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CSSCustomColors client:visible />
  </ExampleWrapper>
</CodeDemo>
