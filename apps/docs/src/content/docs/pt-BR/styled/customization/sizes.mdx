# Tamanhos

O pacote `styled` inclui 4 variantes de tamanho integradas que escalam tipografia, espaçamento, dimensões da célula e dimensões do gatilho proporcionalmente.

## Tamanhos Disponíveis

| Tamanho   | Tamanho da Célula | Fonte Base | Largura Mín. do Gatilho |
| --------- | ----------------- | ---------- | ----------------------- |
| `small`   | 28px              | 11px       | 160px                   |
| `medium`  | 36px              | 13px       | 200px                   |
| `large`   | 44px              | 14px       | 240px                   |
| `x-large` | 52px              | 15px       | 280px                   |

`medium` é o padrão.

## Uso

Passe a prop `size` para qualquer componente de seletor:

<CodeDemo source={SizesSource}>
  <ExampleWrapper variant="styled" client:load>
    <Sizes client:load />
  </ExampleWrapper>
</CodeDemo>

## Tamanhos Personalizados

Crie seu próprio tamanho definindo uma classe CSS que sobrescreve as variáveis de dimensionamento:

```css
.rdrp-size-custom {
  --rdrp-cell-size: 40px;
  --rdrp-nav-btn-size: 30px;
  --rdrp-font-size-base: 14px;
  --rdrp-font-size-md: 15px;
  --rdrp-trigger-min-width: 220px;
  --rdrp-popup-padding: 18px;
}
```

Depois, aplique-o através de um wrapper:

```tsx
<div className="rdrp-size-custom">
  <DatePicker value={value} onChange={setValue} />
</div>
```

Ou com a API de Componentes Compostos, adicione a classe ao Root:

```tsx
<DatePicker.Root value={value} onChange={setValue} className="rdrp-size-custom">
  {/* ... */}
</DatePicker.Root>
```

Você também pode sobrescrever variáveis de dimensionamento através de estilos embutidos:

<CodeDemo source={CustomSizeViaCSSSource}>
  <ExampleWrapper variant="styled" client:load>
    <CustomSizeViaCSS client:load />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip">
  O tamanho `medium` é o padrão e não aplica nenhuma sobrescrita. As variáveis de base em
  `variables.css` são os valores do tamanho médio.
</Admonition>
