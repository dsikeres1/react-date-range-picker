# Rodapé Personalizado

Construa um rodapé completamente personalizado com seus próprios botões e layout, ou remova partes inteiramente para um seletor minimalista.

## Rodapé Reorganizado

Reorganize os botões do rodapé ou adicione novos como o `TodayButton`. O componente composto `DateTimePicker` inclui uma `TimeSection` com um `TimePanel` para seleção de tempo, e o `Footer` atua como um contêiner para os botões de ação.

<CodeDemo source={CompoundCustomFooterSource}>
  <ExampleWrapper variant="styled" client:load>
    <CompoundCustomFooter client:load />
  </ExampleWrapper>
</CodeDemo>

Você também pode pular totalmente os botões de ação integrados e usar seus próprios elementos dentro do `Footer`:

```tsx
<DatePicker.Footer>
  <button onClick={() => setValue(new Date())}>Set Today</button>
  <button onClick={() => setValue(null)}>Clear</button>
</DatePicker.Footer>
```

## Layout Minimalista

Omita quaisquer partes que você não precise. Por exemplo, um calendário embutido sem cabeçalho, rodapé ou gatilho:

<CodeDemo source={CompoundNoFooterSource}>
  <ExampleWrapper variant="styled" client:load>
    <CompoundNoFooter client:load />
  </ExampleWrapper>
</CodeDemo>

Ou um seletor sem o rodapé:

```tsx
<DatePicker.Root value={value} onChange={setValue}>
  <DatePicker.Trigger />
  <DatePicker.Content>
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>
```

## Adicionando Nomes de Classe Personalizados

Cada parte do componente composto aceita uma propriedade `className` que é mesclada com as classes CSS estilizadas padrão:

```tsx
<DatePicker.Root value={value} onChange={setValue} className="my-picker">
  <DatePicker.Trigger className="my-trigger" />
  <DatePicker.Content className="my-content">
    <DatePicker.Header className="my-header">
      <DatePicker.PrevButton className="my-nav" />
      <DatePicker.Title className="my-title" />
      <DatePicker.NextButton className="my-nav" />
    </DatePicker.Header>
    <DatePicker.Grid className="my-grid" />
  </DatePicker.Content>
</DatePicker.Root>
```
