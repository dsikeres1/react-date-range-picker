# Tipografia

O pacote `styled` utiliza uma **pilha de fontes do sistema** por padrão:

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

Isso renderiza com a fonte nativa do sistema operacional em cada plataforma (San Francisco no macOS, Segoe UI no Windows, Roboto no Android). Fontes do sistema oferecem bom suporte a múltiplos idiomas, já que o sistema operacional lida automaticamente com fallbacks para CJK, árabe e outros scripts.

Para substituir, defina `--rdrp-font-family` em qualquer elemento ancestral — o estilo será aplicado em cascata ao seletor.

## Usando Fontes da Web (Noto Sans)

Para uma renderização consistente entre plataformas ou em projetos de i18n, carregue uma fonte da web como a [Noto Sans](https://fonts.google.com/noto) e substitua a variável:

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/styles.css";

// 1. Carregue a Noto Sans no seu HTML ou CSS:
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. Substitua a variável da fonte:
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## Tamanho e Peso da Fonte

Ajuste a densidade da tipografia substituindo as variáveis de tamanho e peso:

| Variável                      | Padrão | Descrição                  |
| ----------------------------- | ------ | -------------------------- |
| `--rdrp-font-size-xs`         | `11px` | Texto menor                |
| `--rdrp-font-size-sm`         | `12px` | Rótulos dos dias da semana |
| `--rdrp-font-size-base`       | `13px` | Células de dia, botões     |
| `--rdrp-font-size-md`         | `14px` | Gatilho, itens de tempo    |
| `--rdrp-font-size-lg`         | `15px` | Título do mês              |
| `--rdrp-font-weight-normal`   | `400`  | Peso base                  |
| `--rdrp-font-weight-medium`   | `500`  | Ênfase média               |
| `--rdrp-font-weight-semibold` | `600`  | Dias selecionados, títulos |
| `--rdrp-font-weight-bold`     | `700`  | Indicador de hoje          |
