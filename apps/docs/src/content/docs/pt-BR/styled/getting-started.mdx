# Começando com Styled

O pacote styled fornece componentes de seletor de data pré-estilizados com CSS embutido. Nenhum framework CSS é necessário.

## Instalação

<PackageInstall pkg="react-date-range-picker-styled" />

Após a instalação, importe o CSS no ponto de entrada do seu aplicativo:

```tsx
import "react-date-range-picker-styled/rdrp-styles.css";
```

## Início Rápido

    <StyledBasic client:load />
  </ExampleWrapper>
</CodeDemo>
