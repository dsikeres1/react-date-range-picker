# 颜色

Tailwind v3 包使用 **硬编码的 Tailwind 功能类** (例如 `bg-sky-500`) 而不是 CSS 变量。要切换颜色，请使用 Compound API 并动态覆盖日期类名：

<Admonition type="info">
  与使用 CSS 变量的 styled 包不同，Tailwind v3 需要 Compound API
  在运行时覆盖颜色。有关更多详细信息，请参阅
  [主题覆盖](/docs/tailwind3/customization/theme-override)。
</Admonition>
