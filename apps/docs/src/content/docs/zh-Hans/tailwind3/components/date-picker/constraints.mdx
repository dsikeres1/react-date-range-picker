# 约束

## 必填

当设置 `required` 后，清除按钮将被禁用，且值不能为空。

  <ExampleWrapper variant="tailwind3" client:visible>
    <Required client:visible />
  </ExampleWrapper>
</CodeDemo>

## 最小和最大日期

使用 `minDate` 和 `maxDate` 将可选日期限制在特定范围内。

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinMaxDate client:visible />
  </ExampleWrapper>
</CodeDemo>
