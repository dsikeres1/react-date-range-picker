# 约束

## 最小和最大天数

使用 `minDays` 和 `maxDays`（含）限制范围长度。用户无法确认短于 `minDays` 或长于 `maxDays` 的范围。

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinMaxDays client:visible />
  </ExampleWrapper>
</CodeDemo>

## 禁用日期

使用 `isDateUnavailable` 动态禁用特定日期。禁用的日期不能被选为范围的开始或结束。

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <DisabledDates client:visible />
  </ExampleWrapper>
</CodeDemo>

## 允许单日

默认情况下，允许开始和结束日期相同。设置 `allowSingleDateInRange={false}` 以要求至少两个不同的日期。

  <ExampleWrapper variant="tailwind3" client:visible>
    <AllowSingleDate client:visible />
  </ExampleWrapper>
</CodeDemo>
