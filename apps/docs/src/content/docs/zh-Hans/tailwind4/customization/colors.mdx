# 颜色

通过在包装元素上覆盖 CSS 自定义属性，在运行时切换颜色主题。由于 Tailwind v4 包从语义化令牌中读取所有颜色，更改底层的 CSS 变量会立即为整个选择器重塑品牌。

Tailwind v4 包对语义化令牌使用 **oklch** 格式。覆盖 `--color-primary` 和 `--color-primary-foreground` 来为选择器重塑品牌：

<CodeDemo source={CompoundColorThemeSource}>

    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>
