# 排版

Tailwind v4 包使用 Tailwind 的默认 `font-sans` 类 (`ui-sans-serif, system-ui, sans-serif, ...`)。由于 CSS 的 `font-family` 是继承的，你可以通过将选择器包装在一个带有 `fontFamily` 样式的容器中来覆盖其字体。包装器的字体会层叠到所有选择器元素中。

## 字体家族

包裹选择器并在包装器上设置 `fontFamily` 来切换字体：

<CodeDemo source={CompoundCustomFontSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <CompoundCustomFont client:load />
  </ExampleWrapper>
</CodeDemo>

## 使用网页字体 (Noto Sans)

在运行时加载一个网页字体，并通过一个包装器应用它。本示例从 Google Fonts 加载 [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans)：

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <CompoundNotoSans client:load />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="生产环境用法">
在实际项目中，应通过 HTML `<head>` 中的 `<link>` 标签或通过 CSS 打包工具加载字体，而不是在运行时注入。此处的 `useEffect` 方法仅用于演示目的。
</Admonition>
