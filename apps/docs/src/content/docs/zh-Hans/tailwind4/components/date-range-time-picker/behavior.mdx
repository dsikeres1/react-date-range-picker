# 行为

## 选择后关闭

启用 `shouldCloseOnSelect` 后，当点击预设时，选择器会自动确认并关闭。

<Admonition type="info">
  对于启用了时间选择的选择器，`shouldCloseOnSelect` 仅适用于点击预设 — 而不是点击日期，
  因为用户仍需要设置时间。
</Admonition>
