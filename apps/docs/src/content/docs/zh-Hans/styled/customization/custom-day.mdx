# 自定义日期单元格

使用 `Day` 组件的渲染属性 (render prop) 来自定义每个日期单元格的外观。这对于为单个日期添加事件指示器、工具提示或完全自定义的样式非常有用。

## Day 渲染属性

| 属性             | 类型         | 描述                       |
| ---------------- | ------------ | -------------------------- |
| `date`           | `Date`       | 日期对象                   |
| `day`            | `number`     | 月份中的第几天 (1-31)      |
| `isToday`        | `boolean`    | 是否是今天                 |
| `isSelected`     | `boolean`    | 此日期是否被选中           |
| `isDisabled`     | `boolean`    | 此日期是否被禁用           |
| `isInRange`      | `boolean`    | 此日期是否在选定范围内     |
| `isRangeStart`   | `boolean`    | 是否是范围的起始点         |
| `isRangeEnd`     | `boolean`    | 是否是范围的结束点         |
| `isInHoverRange` | `boolean`    | 此日期是否在悬停预览范围内 |
| `isHoverTarget`  | `boolean`    | 是否是悬停的终点           |
| `isOutsideDay`   | `boolean`    | 此日期是否属于相邻月份     |
| `isHighlighted`  | `boolean`    | 此日期是否有一个高亮圆点   |
| `isFocused`      | `boolean`    | 此日期是否具有键盘焦点     |
| `onClick`        | `() => void` | 点击事件处理程序           |
| `onMouseEnter`   | `() => void` | 鼠标进入事件处理程序       |
| `onMouseLeave`   | `() => void` | 鼠标离开事件处理程序       |
