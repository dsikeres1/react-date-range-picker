# 颜色

通过覆盖包装器元素上的 CSS 自定义属性，在运行时切换颜色主题。 `styled` 包使用 `--rdrp-color-primary`、`--rdrp-color-primary-hover`、`--rdrp-color-primary-light` 和 `--rdrp-color-primary-lighter` 作为所有主要强调色。

## 内联主题切换器

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

## 弹出式主题切换器

同样的方法也适用于弹出式选择器：

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorThemePopup client:visible />
  </ExampleWrapper>
</CodeDemo>

## CSS 变量内联样式

你也可以通过包装器元素上的 `style` 属性直接覆盖颜色：

<CodeDemo source={CSSCustomColorsSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CSSCustomColors client:visible />
  </ExampleWrapper>
</CodeDemo>
