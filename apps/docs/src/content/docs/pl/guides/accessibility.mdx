# Dostępność

Wszystkie komponenty do wyboru daty są dostarczane z wbudowanym wsparciem dostępności. Nawigacja za pomocą klawiatury, atrybuty ARIA i zarządzanie focusem działają od razu po wyjęciu z pudełka w każdym wariancie — Styled, Tailwind v3/v4 i Headless.

Standardowe użycie nie wymaga dodatkowej konfiguracji. Ta strona dokumentuje, co jest zawarte, abyś mógł audytować, rozszerzać lub powielać wzorce w niestandardowych interfejsach użytkownika.
