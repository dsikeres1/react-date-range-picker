# Zachowanie

## Zamykanie po wybraniu

Gdy opcja `shouldCloseOnSelect` jest włączona, picker automatycznie zatwierdza i zamyka się po kliknięciu presetu.

<Admonition type="info">
  Dla pickerów z włączoną obsługą czasu, `shouldCloseOnSelect` dotyczy tylko kliknięć presetów — nie
  kliknięć dat, ponieważ użytkownik wciąż musi ustawić czas.
</Admonition>
