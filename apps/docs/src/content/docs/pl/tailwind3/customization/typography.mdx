# Typografia

Paczka Tailwind v3 używa domyślnej klasy Tailwind `font-sans` (`ui-sans-serif, system-ui, sans-serif, ...`). Ponieważ właściwość CSS `font-family` jest dziedziczona, możesz nadpisać czcionkę selektora, umieszczając go w kontenerze ze stylem `fontFamily`. Czcionka kontenera zostanie zastosowana do wszystkich elementów selektora.

## Krój czcionki

Owiń selektor i ustaw `fontFamily` na kontenerze, aby zmienić czcionki:

    <CompoundCustomFont client:visible />

</CodeDemo>

## Używanie czcionek internetowych (Noto Sans)

Załaduj czcionkę internetową w czasie wykonywania i zastosuj ją za pomocą kontenera. Ten przykład ładuje [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) z Google Fonts:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="Użycie w produkcji">
W prawdziwym projekcie załaduj czcionkę za pomocą tagu `<link>` w sekcji `<head>` pliku HTML lub za pośrednictwem narzędzia do kompilacji CSS, zamiast wstrzykiwać ją w czasie wykonywania. Użyte tutaj podejście z `useEffect` służy wyłącznie celom demonstracyjnym.
</Admonition>
