# Kolory

Paczka Tailwind v3 używa **zakodowanych na stałe klas użytkowych Tailwind** (np. `bg-sky-500`) zamiast zmiennych CSS. Aby zmienić kolory, użyj Compound API i dynamicznie nadpisuj nazwy klas dni:

<Admonition type="info">
  W przeciwieństwie do paczki `styled` (która używa zmiennych CSS), Tailwind v3 wymaga użycia
  Compound API do nadpisywania kolorów w czasie rzeczywistym. Zobacz [Nadpisywanie
  motywu](/docs/tailwind3/customization/theme-override), aby uzyskać więcej szczegółów.
</Admonition>
