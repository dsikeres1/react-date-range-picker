# Kolory

Przełączaj motywy kolorów w czasie rzeczywistym, nadpisując niestandardowe właściwości CSS na elemencie otaczającym. Ponieważ pakiet Tailwind v4 odczytuje wszystkie kolory z semantycznych tokenów, zmiana podstawowych zmiennych CSS natychmiastowo zmienia wygląd całego pickera.

Pakiet Tailwind v4 używa formatu **oklch** dla semantycznych tokenów. Nadpisz `--color-primary` i `--color-primary-foreground`, aby zmienić wygląd pickera:

  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>
