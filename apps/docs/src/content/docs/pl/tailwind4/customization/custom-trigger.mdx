# Niestandardowy Wyzwalacz

Zastąp domyślny przycisk wyzwalający własnym elementem, używając render prop `Trigger`. Kiedy przekażesz funkcję jako dziecko do `DatePicker.Trigger`, otrzymasz stan i wywołania zwrotne, aby zbudować dowolny interfejs wyzwalacza.

    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Render Props Wyzwalacza

| Prop           | Typ              | Opis                                               |
| -------------- | ---------------- | -------------------------------------------------- |
| `displayValue` | `string`         | Sformatowana wybrana data                          |
| `placeholder`  | `string`         | Tekst zastępczy                                    |
| `hasValue`     | `boolean`        | Czy data jest wybrana                              |
| `isOpen`       | `boolean`        | Czy okno podręczne jest otwarte                    |
| `onToggle`     | `() => void`     | Przełączanie otwierania/zamykania okna podręcznego |
| `onClear`      | `() => void`     | Wyczyść zaznaczenie                                |
| `locale`       | `Locale`         | Aktualny obiekt lokalizacji                        |
| `triggerRef`   | `(node) => void` | Wywołanie zwrotne ref dla pozycjonowania           |
