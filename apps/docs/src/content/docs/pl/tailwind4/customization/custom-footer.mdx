# Niestandardowa stopka

## Zmieniony układ stopki

Zmień kolejność przycisków w stopce lub dodaj nowe, takie jak `TodayButton`. Ponieważ każdy przycisk jest niezależną częścią składową, masz pełną kontrolę nad ich kolejnością i układem.

## Nadpisywanie klas

Każda część składowa akceptuje właściwość `className`. Pakiet dla Tailwind v4 używa `cn()` (narzędzia łączącego `clsx` i `twMerge`), aby połączyć Twoje klasy z domyślnymi, dzięki czemu konflikty w stylach są rozwiązywane poprawnie:

```tsx
import { DatePicker } from "react-date-range-picker-tailwind4";

<DatePicker.Root value={value} onChange={setValue}>
  {/* Nadpisz szerokość i dopełnienie przycisku */}
  <DatePicker.Trigger className="min-w-[300px] px-4 py-3" />
  <DatePicker.Content className="rounded-2xl shadow-xl">
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title className="text-lg" />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>;
```
