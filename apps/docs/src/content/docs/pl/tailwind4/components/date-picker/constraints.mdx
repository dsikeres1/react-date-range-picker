# Ograniczenia

## Wymagane

Gdy `required` jest ustawione, przycisk czyszczenia jest wyłączony, a wartość nie może być pusta.

  <ExampleWrapper variant="tailwind4" client:load>
    <Required client:load />
  </ExampleWrapper>
</CodeDemo>

## Minimalna i maksymalna data

Ogranicz wybieralne daty do określonego zakresu za pomocą `minDate` i `maxDate`.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <MinMaxDate client:load />
  </ExampleWrapper>
</CodeDemo>
