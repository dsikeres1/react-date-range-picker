# Zachowanie

## Zamykanie przy wyborze

Gdy opcja `shouldCloseOnSelect` jest włączona, picker automatycznie zatwierdza wybór i zamyka się po kliknięciu presetu.

<Admonition type="info">
  W przypadku pickerów z włączoną obsługą czasu, opcja `shouldCloseOnSelect` dotyczy tylko kliknięć
  presetów — a nie kliknięć dat, ponieważ użytkownik wciąż musi ustawić czas.
</Admonition>
