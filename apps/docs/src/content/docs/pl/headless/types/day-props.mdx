# DayProps

Obliczone flagi dla każdej komórki, zwracane przez `getDayProps()`. Opisują wizualny i interaktywny stan pojedynczej komórki dnia w kalendarzu.

```tsx
import type { DayProps } from "react-date-range-picker-headless";
```

## Pola

| Pole   | Typ    | Opis                           |
| ------ | ------ | ------------------------------ |
| `date` | `Date` | Obiekt `Date` dla tej komórki. |
