# Ograniczenia

## Minimalna i maksymalna liczba dni

Ogranicz długość zakresu za pomocą `minDays` i `maxDays` (włącznie). Użytkownicy nie mogą zatwierdzić zakresu krótszego niż `minDays` lub dłuższego niż `maxDays`.

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="styled" client:load>
    <MinMaxDays client:load />
  </ExampleWrapper>
</CodeDemo>

## Wyłączone daty

Użyj `isDateUnavailable`, aby dynamicznie wyłączać określone daty. Wyłączone daty nie mogą być wybrane jako początek ani koniec zakresu.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="styled" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## Zezwalaj na pojedynczą datę

Domyślnie dozwolony jest zakres, w którym data początkowa jest równa końcowej. Ustaw `allowSingleDateInRange={false}`, aby wymagać co najmniej dwóch różnych dat.
