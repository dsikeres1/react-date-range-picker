# Kolory

Zmieniaj motyw kolorystyczny w czasie rzeczywistym, nadpisując niestandardowe właściwości CSS na elemencie otaczającym. Pakiet `styled` używa `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light`, oraz `--rdrp-color-primary-lighter` dla wszystkich głównych kolorów akcentujących.

## Przełącznik Motywu w Linii

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

  <ExampleWrapper variant="styled" client:visible>

## Styl Liniowy Zmiennych CSS

Możesz także nadpisać kolory bezpośrednio za pomocą atrybutu `style` na elemencie otaczającym:

  <ExampleWrapper variant="styled" client:visible>
    <CSSCustomColors client:visible />
  </ExampleWrapper>
