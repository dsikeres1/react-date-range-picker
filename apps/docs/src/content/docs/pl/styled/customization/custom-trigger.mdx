# Niestandardowy Wyzwalacz

Zastąp domyślny wyzwalacz własnym przyciskiem lub polem wejściowym. Komponent `Trigger` akceptuje render prop, który daje pełną kontrolę nad wyglądem i zachowaniem wyzwalacza.

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="styled" client:load>
    <CompoundCustomTrigger client:load />
  </ExampleWrapper>
</CodeDemo>

## Render Props dla Wyzwalacza

| Prop           | Typ              | Opis                                 |
| -------------- | ---------------- | ------------------------------------ |
| `displayValue` | `string`         | Sformatowana wybrana data            |
| `placeholder`  | `string`         | Tekst zastępczy                      |
| `hasValue`     | `boolean`        | Czy data jest wybrana                |
| `isOpen`       | `boolean`        | Czy okienko jest otwarte             |
| `onToggle`     | `() => void`     | Przełącz otwarcie/zamknięcie okienka |
| `onClear`      | `() => void`     | Wyczyść zaznaczenie                  |
| `locale`       | `Locale`         | Aktualny obiekt locale               |
| `triggerRef`   | `(node) => void` | Callback ref do pozycjonowania       |
