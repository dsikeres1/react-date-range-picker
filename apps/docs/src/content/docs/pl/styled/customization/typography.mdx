# Typografia

Pakiet `styled` domyślnie używa **systemowego stosu czcionek**:

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

Renderuje się to za pomocą natywnej czcionki systemu operacyjnego na każdej platformie (San Francisco na macOS, Segoe UI na Windows, Roboto na Android). Czcionki systemowe zapewniają dobre wsparcie dla wielu języków, ponieważ system operacyjny automatycznie obsługuje zastępstwa dla CJK, arabskiego i innych skryptów.

Aby nadpisać, ustaw `--rdrp-font-family` na dowolnym elemencie nadrzędnym — kaskadowo przeniesie się to do pickera.

## Używanie Czcionek Internetowych (Noto Sans)

Dla spójnego renderowania na różnych platformach lub w projektach i18n, załaduj czcionkę internetową, taką jak [Noto Sans](https://fonts.google.com/noto), i nadpisz zmienną:

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/styles.css";

// 1. Załaduj Noto Sans w swoim HTML lub CSS:
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. Nadpisz zmienną czcionki:
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## Rozmiar i Waga Czcionki

Dostosuj gęstość typografii, nadpisując zmienne rozmiaru i wagi:

| Zmienna                       | Domyślna | Opis                       |
| ----------------------------- | -------- | -------------------------- |
| `--rdrp-font-size-xs`         | `11px`   | Najmniejszy tekst          |
| `--rdrp-font-size-sm`         | `12px`   | Etykiety dni tygodnia      |
| `--rdrp-font-size-base`       | `13px`   | Komórki dni, przyciski     |
| `--rdrp-font-size-md`         | `14px`   | Wyzwalacz, elementy czasu  |
| `--rdrp-font-size-lg`         | `15px`   | Tytuł miesiąca             |
| `--rdrp-font-weight-normal`   | `400`    | Podstawowa waga            |
| `--rdrp-font-weight-medium`   | `500`    | Średnie wyróżnienie        |
| `--rdrp-font-weight-semibold` | `600`    | Wybrane dni, tytuły        |
| `--rdrp-font-weight-bold`     | `700`    | Wskaźnik dzisiejszego dnia |
