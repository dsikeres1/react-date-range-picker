# สี

สลับธีมสีขณะรันไทม์โดยการลบล้างคุณสมบัติ CSS custom properties บนอิลิเมนต์ครอบ แพ็คเกจ styled ใช้ `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light`, และ `--rdrp-color-primary-lighter` สำหรับสีเน้นหลักทั้งหมด
