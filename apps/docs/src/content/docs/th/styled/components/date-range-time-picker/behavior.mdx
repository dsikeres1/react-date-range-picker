# พฤติกรรม

## ปิดเมื่อเลือก

เมื่อเปิดใช้งาน `shouldCloseOnSelect` ตัวเลือกจะยืนยันและปิดโดยอัตโนมัติเมื่อมีการคลิกค่าที่ตั้งไว้ล่วงหน้า

<Admonition type="info">
  สำหรับตัวเลือกที่เปิดใช้งานเวลา `shouldCloseOnSelect` จะใช้กับการคลิกค่าที่ตั้งไว้ล่วงหน้าเท่านั้น
  — ไม่ใช่การคลิกวันที่ เนื่องจากผู้ใช้ยังคงต้องตั้งเวลา
</Admonition>
