# เครื่องมือเกี่ยวกับปฏิทิน

ฟังก์ชันสำหรับสร้างข้อมูลตารางปฏิทิน

```tsx
import {
  generateCalendarDays,
  daysToWeeks,
  buildCalendarMonth,
} from "react-date-range-picker-headless";
```

## generateCalendarDays

สร้างอาร์เรย์แบบแบนจำนวน 42 ช่อง (6 สัปดาห์ x 7 วัน) สำหรับเดือนที่กำหนด
