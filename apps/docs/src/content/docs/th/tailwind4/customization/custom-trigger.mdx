# ทริกเกอร์แบบกำหนดเอง

แทนที่ปุ่มทริกเกอร์เริ่มต้นด้วยองค์ประกอบของคุณเองโดยใช้ render prop ชื่อ `Trigger` เมื่อคุณส่งฟังก์ชันเป็น child ให้กับ `DatePicker.Trigger` คุณจะได้รับ state และ callbacks เพื่อสร้าง UI ทริกเกอร์ที่คุณต้องการ

## Render Props ของทริกเกอร์

| Prop           | ประเภท           | คำอธิบาย                         |
| -------------- | ---------------- | -------------------------------- |
| `displayValue` | `string`         | วันที่ที่เลือกซึ่งจัดรูปแบบแล้ว  |
| `placeholder`  | `string`         | ข้อความ Placeholder              |
| `hasValue`     | `boolean`        | ระบุว่ามีการเลือกวันที่หรือไม่   |
| `isOpen`       | `boolean`        | ระบุว่าป๊อปอัปเปิดอยู่หรือไม่    |
| `onToggle`     | `() => void`     | สลับการเปิด/ปิดป๊อปอัป           |
| `onClear`      | `() => void`     | ล้างค่าที่เลือก                  |
| `locale`       | `Locale`         | อ็อบเจกต์ locale ปัจจุบัน        |
| `triggerRef`   | `(node) => void` | Ref callback สำหรับการจัดตำแหน่ง |
