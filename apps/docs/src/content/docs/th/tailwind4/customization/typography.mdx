# การพิมพ์

แพ็คเกจ Tailwind v4 ใช้คลาส `font-sans` เริ่มต้นของ Tailwind (`ui-sans-serif, system-ui, sans-serif, ...`) เนื่องจาก CSS `font-family` ถูกสืบทอด คุณสามารถแทนที่ฟอนต์ของตัวเลือกได้โดยการห่อหุ้มด้วยคอนเทนเนอร์ที่มีสไตล์ `fontFamily` ฟอนต์ของคอนเทนเนอร์จะถูกส่งต่อไปยังองค์ประกอบทั้งหมดของตัวเลือก

## รูปแบบฟอนต์

ห่อตัวเลือกและตั้งค่า `fontFamily` บนคอนเทนเนอร์เพื่อเปลี่ยนฟอนต์:

## การใช้เว็บฟอนต์ (Noto Sans)

โหลดเว็บฟอนต์ขณะรันไทม์และนำไปใช้ผ่านคอนเทนเนอร์ ตัวอย่างนี้โหลด [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) จาก Google Fonts:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <CompoundNotoSans client:load />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="การใช้งานจริง">
ในโปรเจกต์จริง ควรโหลดฟอนต์ผ่านแท็ก `<link>` ใน `<head>` ของ HTML หรือผ่าน CSS bundler แทนที่จะฉีดเข้ามาขณะรันไtime วิธีการใช้ `useEffect` ที่นี่มีไว้เพื่อการสาธิตเท่านั้น
</Admonition>
