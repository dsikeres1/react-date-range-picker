# สี

สลับธีมสีขณะรันไทม์โดยการลบล้างคุณสมบัติ CSS custom properties บนองค์ประกอบ wrapper เนื่องจากแพ็คเกจ Tailwind v4 อ่านค่าสีทั้งหมดจาก semantic tokens การเปลี่ยนตัวแปร CSS พื้นฐานจะรีแบรนด์ picker ทั้งหมดทันที

แพ็คเกจ Tailwind v4 ใช้รูปแบบ **oklch** สำหรับ semantic tokens ลบล้าง `--color-primary` และ `--color-primary-foreground` เพื่อรีแบรนด์ picker:

</CodeDemo>
