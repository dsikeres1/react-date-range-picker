# โหมดมืด

แพ็คเกจ Tailwind v3 รองรับโหมดมืดอย่างเต็มรูปแบบโดยใช้ `dark:` variant ของ Tailwind ทุกองค์ประกอบของคอมโพเนนต์มีการกำหนดคลาสสำหรับโหมดมืดไว้อย่างชัดเจน

## วิธีการทำงาน

ธีม Tailwind v3 ใช้คลาสสีที่ระบุเจาะจงพร้อมกับ `dark:` variant อยู่ตลอด ตัวอย่างเช่น:

```
bg-white dark:bg-slate-950
text-slate-900 dark:text-slate-50
border-slate-200 dark:border-slate-800
```

โหมดมืดจะทำงานเมื่อ `dark:` variant ของ Tailwind ถูกกระตุ้น ซึ่งขึ้นอยู่กับกลยุทธ์ `darkMode` ในการตั้งค่า Tailwind ของคุณ

## การตั้งค่า Tailwind

### กลยุทธ์ Class (แนะนำ)

ตั้งค่า `darkMode: "class"` ในไฟล์คอนฟิก Tailwind ของคุณ โหมดมืดจะทำงานเมื่อมีคลาส `dark` อยู่บน `<html>` หรือองค์ประกอบแม่:

```js
// tailwind.config.js
module.exports = {
  darkMode: "class",
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
};
```

สลับโหมดมืด:

```tsx
function App() {
  const [dark, setDark] = useState(false);

  return (
    <div className={dark ? "dark" : ""}>
      <button onClick={() => setDark(!dark)}>Toggle Theme</button>
      <DatePicker value={value} onChange={setValue} />
    </div>
  );
}
```

หรือใช้คลาสกับ `<html>`:

```tsx
document.documentElement.classList.toggle("dark", isDark);
```

### กลยุทธ์ Selector

Tailwind v3.4+ รองรับ `darkMode: "selector"` ซึ่งอนุญาตให้ใช้ `[data-theme="dark"]` หรือ selector ที่กำหนดเองได้:

```js
// tailwind.config.js
module.exports = {
  darkMode: ["selector", '[data-theme="dark"]'],
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
};
```

```tsx
<div data-theme={dark ? "dark" : "light"}>
  <DatePicker value={value} onChange={setValue} />
</div>
```

### กลยุทธ์ Media

ตั้งค่า `darkMode: "media"` (หรือเว้นไว้ — นี่คือค่าเริ่มต้น) เพื่อให้เป็นไปตามการตั้งค่าของระบบปฏิบัติการ:

```js
// tailwind.config.js
module.exports = {
  darkMode: "media",
  // ...
};
```

ไม่จำเป็นต้องตั้งค่าเพิ่มเติม ตัวเลือกวันที่จะทำงานตาม `prefers-color-scheme`

## คู่สี

นี่คือคู่สีสำหรับโหมดสว่าง/มืดที่ใช้ทั่วทั้งตัวเลือกวันที่:

| องค์ประกอบ              | โหมดสว่าง           | โหมดมืด                      |
| ----------------------- | ------------------- | ---------------------------- |
| พื้นหลัง                | `bg-white`          | `dark:bg-slate-950`          |
| ข้อความ                 | `text-slate-900`    | `dark:text-slate-50`         |
| ข้อความสีจาง            | `text-slate-500`    | `dark:text-slate-400`        |
| เส้นขอบ                 | `border-slate-200`  | `dark:border-slate-800`      |
| พื้นหลังเมื่อวางเมาส์   | `bg-slate-100`      | `dark:bg-slate-800`          |
| สีหลัก                  | `bg-sky-500`        | `dark:bg-sky-500`            |
| ข้อความสีหลัก           | `text-sky-500`      | `dark:text-sky-400`          |
| พื้นหลังช่วงวันที่      | `bg-sky-50`         | `dark:bg-sky-950/50`         |
| ช่วงวันที่เมื่อวางเมาส์ | `bg-sky-50/50`      | `dark:bg-sky-950/30`         |
| วงแหวนโฟกัส             | `ring-sky-500`      | `ring-sky-500`               |
| ออฟเซ็ตวงแหวน           | `ring-offset-white` | `dark:ring-offset-slate-950` |

## การผนวกรวมกับ Docusaurus

หากคุณใช้ Docusaurus (ซึ่งใช้ `data-theme` บน `<html>`) ให้ตั้งค่ากลยุทธ์ selector ของ Tailwind:

```js
// tailwind.config.js
module.exports = {
  darkMode: ["selector", '[data-theme="dark"]'],
  // ...
};
```

Docusaurus จะตั้งค่า `<html data-theme="dark">` เมื่อโหมดมืดทำงาน และ `dark:` variant ของ Tailwind จะทำงานตาม selector นี้

## การผนวกรวมกับ next-themes

```tsx
// app/layout.tsx
import { ThemeProvider } from "next-themes";

export default function RootLayout({ children }) {
  return (
    <html suppressHydrationWarning>
      <body>
        <ThemeProvider attribute="class" defaultTheme="system">
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

`next-themes` จะเพิ่ม/ลบคลาส `dark` บน `<html>` เมื่อตั้งค่า `darkMode: "class"` ในคอนฟิก Tailwind ของคุณ ตัวเลือกวันที่จะสลับโหมดโดยอัตโนมัติ

## การบังคับใช้โหมดที่เฉพาะเจาะจง

หากต้องการบังคับให้ตัวเลือกวันที่ใช้โหมดสว่างหรือมืดเสมอ ให้ครอบด้วยคอนเทนเนอร์ที่มีคลาสที่เหมาะสม:

```tsx
{
  /* สว่างเสมอ — ไม่มี "dark" class ancestor */
}
<div className="">
  <DatePicker value={value} onChange={setValue} />
</div>;

{
  /* มืดเสมอ */
}
<div className="dark">
  <DatePicker value={value} onChange={setValue} />
</div>;
```

<Admonition type="note">
เมื่อใช้ `darkMode: "class"` คลาส `dark` จะต้องอยู่บนองค์ประกอบแม่หรือ `<html>` ตัวเลือกวันที่ไม่ได้จัดการการสลับโหมดมืดด้วยตัวเอง
</Admonition>
