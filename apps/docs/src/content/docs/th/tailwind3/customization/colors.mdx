# สี

แพ็คเกจ Tailwind v3 ใช้ **คลาสยูทิลิตี้ของ Tailwind ที่กำหนดค่าไว้ตายตัว** (เช่น `bg-sky-500`) แทนที่จะเป็นตัวแปร CSS หากต้องการเปลี่ยนสี ให้ใช้ Compound API และลบล้างชื่อคลาสของวันแบบไดนามิก:

<Admonition type="info">
  ซึ่งแตกต่างจากแพ็คเกจ styled (ที่ใช้ตัวแปร CSS), Tailwind v3 ต้องการ Compound API
  เพื่อลบล้างสีในขณะรันไทม์ ดูรายละเอียดเพิ่มเติมที่ [Theme
  Override](/docs/tailwind3/customization/theme-override)
</Admonition>
