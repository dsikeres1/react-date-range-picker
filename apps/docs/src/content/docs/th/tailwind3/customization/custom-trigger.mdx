# ทริกเกอร์แบบกำหนดเอง

แทนที่ทริกเกอร์เริ่มต้นด้วยองค์ประกอบของคุณเองโดยใช้ render prop คอมโพเนนต์ `Trigger` ยอมรับฟังก์ชันลูกที่รับสถานะและ callbacks ช่วยให้คุณสามารถเรนเดอร์ปุ่มหรืออินพุตที่กำหนดเองได้

## Render Props ของทริกเกอร์

| Prop           | ประเภท           | คำอธิบาย                               |
| -------------- | ---------------- | -------------------------------------- |
| `displayValue` | `string`         | วันที่ที่เลือกในรูปแบบที่จัดรูปแบบแล้ว |
| `placeholder`  | `string`         | ข้อความ Placeholder                    |
| `hasValue`     | `boolean`        | สถานะว่ามีการเลือกวันที่หรือไม่        |
| `isOpen`       | `boolean`        | สถานะว่าป๊อปอัปเปิดอยู่หรือไม่         |
| `onToggle`     | `() => void`     | สลับการเปิด/ปิดป๊อปอัป                 |
| `onClear`      | `() => void`     | ล้างค่าที่เลือก                        |
| `locale`       | `Locale`         | อ็อบเจกต์ locale ปัจจุบัน              |
| `triggerRef`   | `(node) => void` | Ref callback สำหรับการจัดตำแหน่ง       |
