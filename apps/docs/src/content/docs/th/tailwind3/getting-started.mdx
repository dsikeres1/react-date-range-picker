# เริ่มต้นใช้งาน Tailwind CSS v3

แพ็คเกจ Tailwind v3 มีคอมโพเนนต์ตัวเลือกวันที่ซึ่งจัดรูปแบบด้วยคลาสยูทิลิตี้ของ Tailwind CSS v3

## การติดตั้ง

<PackageInstall pkg="react-date-range-picker-tailwind3" />

ไม่จำเป็นต้องนำเข้า CSS เพิ่มเติม คอมโพเนนต์ใช้คลาสยูทิลิตี้ของ Tailwind โดยตรง

## การกำหนดค่า Tailwind

เพิ่มแพ็คเกจไปยังพาร์ท `content` ของคุณในไฟล์ `tailwind.config.js` เพื่อให้ Tailwind สามารถสแกนคลาสของคอมโพเนนต์ได้:

```js
// tailwind.config.js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
  // ...
};
```

## มีอะไรบ้าง

| คอมโพเนนต์            | คำอธิบาย                                                 |
| --------------------- | -------------------------------------------------------- |
| `DatePicker`          | การเลือกวันที่เดียวพร้อมปฏิทินป๊อปอัป                    |
| `DateRangePicker`     | การเลือกช่วงวันที่พร้อมปฏิทินคู่และค่าที่ตั้งไว้ล่วงหน้า |
| `DateTimePicker`      | การเลือกวันที่ + เวลา พร้อมแผงเลื่อนเวลา                 |
| `DateRangeTimePicker` | การเลือกช่วงวันที่ + เวลา                                |

คอมโพเนนต์ทั้งหมดรองรับ:

- ควบคุมค่าด้วย `value` / `onChange`
- การนำทางด้วยคีย์บอร์ด
- โหมดมืดผ่าน `dark:` variant
- 4 ขนาด: `small`, `medium`, `large`, `x-large`
- Compound Component API สำหรับการปรับแต่ง

## ประเภทของ TypeScript

ประเภทการกำหนดค่าและข้อมูลทั้งหมดจะถูก re-export จากแพ็คเกจ — ไม่จำเป็นต้องติดตั้งแพ็คเกจ headless แยกต่างหาก:

```tsx
import type {
  Locale,
  DatePickerSize,
  TimeConfig,
  MinuteStep,
  SecondStep,
  TimePrecision,
  HourFormat,
  TimePeriod,
  CaptionLayout,
  WeekDay,
  DateRangePreset,
  CalendarMonth,
  DayProps,
} from "react-date-range-picker-tailwind3";
```

ประเภทเหล่านี้มีประโยชน์สำหรับ:

- **`Locale`** — การตั้งค่า locale แบบกำหนดเอง (ชื่อวันในสัปดาห์, ป้ายกำกับปุ่ม, รูปแบบวันที่)
- **`DatePickerSize`** — การกำหนดประเภทของ `size` prop (`"small"` | `"medium"` | `"large"` | `"x-large"`)
- **`TimeConfig`** — การกำหนดค่าความแม่นยำ, รูปแบบ, และขั้นตอนของตัวเลือกเวลา
- **`DateRangePreset`** — การกำหนดค่าที่ตั้งไว้ล่วงหน้าสำหรับ `DateRangePicker` / `DateRangeTimePicker`
- **`CalendarMonth`**, **`DayProps`** — การเรนเดอร์แบบกำหนดเองด้วย Compound Components

ดูคำจำกัดความประเภทโดยละเอียดได้ที่ [Headless Types](/docs/headless/types/common-types)

## Tailwind Preflight & Reset

แพ็คเกจนี้ถือว่าโปรเจกต์ของคุณเปิดใช้งาน Tailwind CSS Preflight (ซึ่งเป็นค่าเริ่มต้น)
หากสภาพแวดล้อมของคุณปิดใช้งาน Preflight (เช่น Docusaurus, Micro Frontends บางตัว) สไตล์ของปุ่มอาจเสียหายได้
ในกรณีนั้น ให้นำเข้า `rdrp-reset.css` ที่ให้มาเพื่อเริ่มต้นสไตล์ของปุ่มด้วยตนเอง:

```tsx
import "react-date-range-picker-tailwind3/rdrp-reset.css";
```
