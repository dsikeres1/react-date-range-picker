# Màu sắc

Chuyển đổi chủ đề màu sắc trong thời gian chạy bằng cách ghi đè các thuộc tính tùy chỉnh CSS trên một phần tử bao bọc. Vì gói Tailwind v4 đọc tất cả các màu từ các token ngữ nghĩa, việc thay đổi các biến CSS cơ bản sẽ ngay lập tức thay đổi thương hiệu của toàn bộ bộ chọn.

Gói Tailwind v4 sử dụng định dạng **oklch** cho các token ngữ nghĩa. Ghi đè `--color-primary` và `--color-primary-foreground` để thay đổi thương hiệu của bộ chọn:

</CodeDemo>
