# Bắt đầu với Tailwind CSS v3

Gói Tailwind v3 cung cấp các thành phần bộ chọn ngày được tạo kiểu bằng các lớp tiện ích của Tailwind CSS v3.

## Cài đặt

<PackageInstall pkg="react-date-range-picker-tailwind3" />

Không cần nhập thêm CSS. Các thành phần sử dụng trực tiếp các lớp tiện ích của Tailwind.

## Cấu hình Tailwind

Thêm gói vào đường dẫn `content` trong tệp `tailwind.config.js` của bạn để Tailwind có thể quét các lớp của thành phần:

```js
// tailwind.config.js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/react-date-range-picker-tailwind3/dist/**/*.{js,mjs}",
  ],
  // ...
};
```

## Bao gồm những gì

| Thành phần            | Mô tả                                              |
| --------------------- | -------------------------------------------------- |
| `DatePicker`          | Chọn một ngày với lịch bật lên                     |
| `DateRangePicker`     | Chọn khoảng ngày với lịch kép và các cài đặt trước |
| `DateTimePicker`      | Chọn ngày + giờ với bảng cuộn thời gian            |
| `DateRangeTimePicker` | Chọn khoảng ngày + giờ                             |

Tất cả các thành phần đều hỗ trợ:

- Giá trị được kiểm soát với `value` / `onChange`
- Điều hướng bằng bàn phím
- Chế độ tối thông qua biến thể `dark:`
- 4 kích thước: `small`, `medium`, `large`, `x-large`
- API Thành phần Hợp chất để tùy chỉnh

## Các loại TypeScript

Tất cả các loại cấu hình và dữ liệu đều được tái xuất từ gói — không cần cài đặt riêng gói headless:

```tsx
import type {
  Locale,
  DatePickerSize,
  TimeConfig,
  MinuteStep,
  SecondStep,
  TimePrecision,
  HourFormat,
  TimePeriod,
  CaptionLayout,
  WeekDay,
  DateRangePreset,
  CalendarMonth,
  DayProps,
} from "react-date-range-picker-tailwind3";
```

Các loại này hữu ích cho:

- **`Locale`** — Ghi đè bản địa hóa tùy chỉnh (tên các ngày trong tuần, nhãn nút, định dạng ngày)
- **`DatePickerSize`** — Định kiểu cho prop `size` (`"small"` | `"medium"` | `"large"` | `"x-large"`)
- **`TimeConfig`** — Cấu hình độ chính xác, định dạng và các bước của bộ chọn thời gian
- **`DateRangePreset`** — Xác định các cài đặt trước cho `DateRangePicker` / `DateRangeTimePicker`
- **`CalendarMonth`**, **`DayProps`** — Kết xuất tùy chỉnh với các Thành phần Hợp chất

Xem [Các loại Headless](/docs/headless/types/common-types) để biết định nghĩa chi tiết về các loại.

## Tailwind Preflight & Reset

Gói này giả định rằng dự án của bạn đã bật Tailwind CSS Preflight (đây là mặc định).
Nếu môi trường của bạn tắt Preflight (ví dụ: Docusaurus, một số Micro Frontend), các kiểu nút có thể bị hỏng.
Trong trường hợp đó, hãy nhập tệp `reset.css` được cung cấp để khởi tạo thủ công các kiểu nút:

```tsx
import "react-date-range-picker-tailwind3/reset.css";
```
