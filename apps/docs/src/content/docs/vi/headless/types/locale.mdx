# Bản địa hoá

Giao diện cho tất cả các chuỗi văn bản hiển thị cho người dùng. Truyền một `Partial<Locale>` vào bất kỳ hook nào để ghi đè các chuỗi cụ thể -- các trường không được chỉ định sẽ sử dụng giá trị mặc định từ `DEFAULT_LOCALE`.

```tsx
import type { Locale } from "react-date-range-picker-headless";
```
