# Tiện ích Lịch

Các hàm để tạo dữ liệu lưới lịch.

```tsx
import {
  generateCalendarDays,
  daysToWeeks,
  buildCalendarMonth,
} from "react-date-range-picker-headless";
```

## generateCalendarDays

Tạo một mảng phẳng gồm 42 ô (6 tuần x 7 ngày) cho một tháng nhất định.
