# Trigger Tùy chỉnh

Thay thế trigger mặc định bằng nút hoặc ô nhập liệu của riêng bạn. Thành phần `Trigger` chấp nhận một render prop cho phép bạn toàn quyền kiểm soát giao diện và hành vi của trigger.

</CodeDemo>

## Render Props của Trigger

| Prop           | Type             | Mô tả                                |
| -------------- | ---------------- | ------------------------------------ |
| `displayValue` | `string`         | Ngày đã chọn được định dạng          |
| `placeholder`  | `string`         | Văn bản giữ chỗ                      |
| `hasValue`     | `boolean`        | Liệu một ngày đã được chọn hay chưa  |
| `isOpen`       | `boolean`        | Liệu popup có đang mở hay không      |
| `onToggle`     | `() => void`     | Bật/tắt trạng thái mở/đóng của popup |
| `onClear`      | `() => void`     | Xóa lựa chọn                         |
| `locale`       | `Locale`         | Đối tượng locale hiện tại            |
| `triggerRef`   | `(node) => void` | Ref callback để định vị              |
