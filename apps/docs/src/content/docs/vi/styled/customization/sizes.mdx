# Kích thước

Gói styled bao gồm 4 biến thể kích thước tích hợp giúp thay đổi tỷ lệ kiểu chữ, khoảng cách, kích thước ô và kích thước trình kích hoạt một cách tương ứng.

## Các kích thước có sẵn

| Kích thước | Kích thước ô | Font cơ bản | Chiều rộng tối thiểu của Trigger |
| ---------- | ------------ | ----------- | -------------------------------- |
| `small`    | 28px         | 11px        | 160px                            |
| `medium`   | 36px         | 13px        | 200px                            |
| `large`    | 44px         | 14px        | 240px                            |
| `x-large`  | 52px         | 15px        | 280px                            |

`medium` là mặc định.

## Cách sử dụng

Truyền prop `size` cho bất kỳ thành phần picker nào:

## Kích thước tùy chỉnh

Tạo kích thước của riêng bạn bằng cách định nghĩa một lớp CSS ghi đè các biến kích thước:

```css
.rdrp-size-custom {
  --rdrp-cell-size: 40px;
  --rdrp-nav-btn-size: 30px;
  --rdrp-font-size-base: 14px;
  --rdrp-font-size-md: 15px;
  --rdrp-trigger-min-width: 220px;
  --rdrp-popup-padding: 18px;
}
```

Sau đó áp dụng nó thông qua một thành phần bao bọc:

```tsx
<div className="rdrp-size-custom">
  <DatePicker value={value} onChange={setValue} />
</div>
```

Hoặc với API Compound Component, thêm class vào Root:

```tsx
<DatePicker.Root value={value} onChange={setValue} className="rdrp-size-custom">
  {/* ... */}
</DatePicker.Root>
```

Bạn cũng có thể ghi đè các biến kích thước thông qua style nội tuyến:

<Admonition type="tip">
  Kích thước `medium` là mặc định và không áp dụng bất kỳ ghi đè nào. Các biến cơ sở trong
  `variables.css` là các giá trị của medium.
</Admonition>
