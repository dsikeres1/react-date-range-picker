# タイポグラフィ

`styled` パッケージは、デフォルトで **システムフォントスタック** を使用します:

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

これは各プラットフォームのOSネイティブフォントでレンダリングされます（macOSではSan Francisco、WindowsではSegoe UI、AndroidではRoboto）。システムフォントは、OSがCJK、アラビア語、その他のスクリプトのフォールバックを自動的に処理するため、優れた多言語サポートを提供します。

上書きするには、任意の祖先要素に `--rdrp-font-family` を設定します。これはピッカーにカスケードされます。

## Webフォントの使用 (Noto Sans)

プラットフォーム間で一貫したレンダリングやi18nプロジェクトのためには、[Noto Sans](https://fonts.google.com/noto) のようなWebフォントを読み込み、変数を上書きします:

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/styles.css";

// 1. HTMLまたはCSSでNoto Sansを読み込みます:
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. フォント変数を上書きします:
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## フォントサイズとウェイト

サイズとウェイトの変数を上書きして、タイポグラフィの密度を微調整します:

| 変数                          | デフォルト | 説明                     |
| ----------------------------- | ---------- | ------------------------ |
| `--rdrp-font-size-xs`         | `11px`     | 最小のテキスト           |
| `--rdrp-font-size-sm`         | `12px`     | 曜日のラベル             |
| `--rdrp-font-size-base`       | `13px`     | 日付セル、ボタン         |
| `--rdrp-font-size-md`         | `14px`     | トリガー、時刻アイテム   |
| `--rdrp-font-size-lg`         | `15px`     | 月のタイトル             |
| `--rdrp-font-weight-normal`   | `400`      | 基本のウェイト           |
| `--rdrp-font-weight-medium`   | `500`      | 中程度の強調             |
| `--rdrp-font-weight-semibold` | `600`      | 選択された日付、タイトル |
| `--rdrp-font-weight-bold`     | `700`      | 「今日」のインジケーター |
