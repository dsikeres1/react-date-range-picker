# TimePicker

入力トリガーとポップアップパネルを備えたタイムピッカーです。入力をクリックすると、スクロール可能な時間の列があるドロップダウンが開きます。

## 基本的な使い方

<CodeDemo source={BasicSource}>
  <ExampleWrapper variant="styled" client:load>
    <Basic client:load />
  </ExampleWrapper>
</CodeDemo>

## インライン

トリガーやポップアップなしで、タイムパネルをインラインでレンダringします。

<CodeDemo source={InlineSource}>
  <ExampleWrapper variant="styled" client:load>
    <Inline client:load />
  </ExampleWrapper>
</CodeDemo>
