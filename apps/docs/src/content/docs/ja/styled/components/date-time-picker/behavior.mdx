# 動作

## 選択時に閉じる

`shouldCloseOnSelect` が有効な場合、日付をクリックするとピッカーが自動的に確定して閉じます。

  <ExampleWrapper variant="styled" client:visible>
    <ShouldCloseOnSelect client:visible />
  </ExampleWrapper>
</CodeDemo>

## 複数月の表示

    <MultiMonth client:visible />
  </ExampleWrapper>
</CodeDemo>

`name` prop を使用して、ネイティブフォーム送信用の非表示 input を含めることができます。

  この例の送信ボタンのスタイルはカスタムです — ライブラリは日付ピッカーのみを提供します。
</Admonition>
