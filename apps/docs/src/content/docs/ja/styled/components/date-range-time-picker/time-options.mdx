# 時間オプション

`time` propを介して、時間パネルの動作を設定します。

## 秒単位の精度

開始時刻と終了時刻の両方のパネルに秒の列を表示します。

<CodeDemo source={SecondPrecisionSource}>
  <ExampleWrapper variant="styled" client:load>
    <SecondPrecision client:load />
  </ExampleWrapper>
</CodeDemo>

## 12時間形式

開始時刻と終了時刻の両方のパネルで、AM/PMトグル付きの12時間形式に切り替えます。
