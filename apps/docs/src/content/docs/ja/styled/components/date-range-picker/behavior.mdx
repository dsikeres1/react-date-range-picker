# 動作

## 選択時に閉じる

`shouldCloseOnSelect` が有効な場合、終了日を選択するとピッカーが自動的に確定し、閉じます。

<CodeDemo source={ShouldCloseOnSelectSource}>
  <ExampleWrapper variant="styled" client:visible>
    <ShouldCloseOnSelect client:visible />
  </ExampleWrapper>
</CodeDemo>

## 複数月表示

デフォルトの2ヶ月表示より多くの月を並べて表示します。

<CodeDemo source={MultiMonthSource}>
  <ExampleWrapper variant="styled" client:visible>
    <MultiMonth client:visible />
  </ExampleWrapper>
</CodeDemo>

## フォーム連携

`name` prop を使用して、ネイティブフォームの送信用の非表示 input を含めることができます。

  この例の送信ボタンのスタイルはカスタムです — ライブラリは日付ピッカーのみを提供します。
</Admonition>
