# 動作

## 選択時に閉じる

`shouldCloseOnSelect` が有効な場合、プリセットをクリックするとピッカーが自動的に確定して閉じます。

<Admonition type="info">
  時間入力が有効なピッカーの場合、`shouldCloseOnSelect`
  はプリセットのクリックにのみ適用され、日付のクリックには適用されません。これは、ユーザーがまだ時間を設定する必要があるためです。
</Admonition>

  </ExampleWrapper>
</CodeDemo>

  この例の送信ボタンのスタイルはカスタムです — ライブラリが提供するのは日付ピッカーのみです。
