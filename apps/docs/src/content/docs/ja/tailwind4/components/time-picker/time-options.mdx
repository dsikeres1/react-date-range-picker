# 時刻オプション

`time` prop を介して時刻パネルの動作を設定します。

## 12時間形式

AM/PMトグル付きの12時間形式に切り替えます。

<CodeDemo source={TwelveHourSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <TwelveHour client:visible />
  </ExampleWrapper>
</CodeDemo>

## 分のステップ

分の増分ステップを設定します。固定間隔でのスケジューリングに便利です。

<CodeDemo source={MinuteStepSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <MinuteStep client:visible />
  </ExampleWrapper>
</CodeDemo>

## 秒の精度

`precision` を `"second"` に設定して、秒の列を表示します。
