# 動作

## 選択時に閉じる

`shouldCloseOnSelect` が有効な場合、終了日を選択するとピッカーが自動的に確定し閉じます。

<CodeDemo source={ShouldCloseOnSelectSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <ShouldCloseOnSelect client:visible />
  </ExampleWrapper>
</CodeDemo>

## 複数月表示

デフォルトの2ヶ月以上を並べて表示します。

<CodeDemo source={MultiMonthSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <MultiMonth client:visible />
  </ExampleWrapper>
</CodeDemo>

## フォーム連携

`name` prop を使用して、ネイティブのフォーム送信用の非表示入力を含めることができます。

  この例の送信ボタンのスタイルはカスタムです — ライブラリが提供するのは日付ピッカーのみです。
</Admonition>
