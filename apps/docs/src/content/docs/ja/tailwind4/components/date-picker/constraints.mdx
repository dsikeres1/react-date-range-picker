# 制約

## 必須

`required` が設定されている場合、クリアボタンは無効になり、値を空にすることはできません。

<CodeDemo source={RequiredSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <Required client:load />
  </ExampleWrapper>
</CodeDemo>

## 最小日と最大日

`minDate` と `maxDate` を使用して、選択可能な日付を特定の日付範囲に制限します。

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <MinMaxDate client:load />
  </ExampleWrapper>
</CodeDemo>

## 無効な日付

`isDateUnavailable` を使用して、特定の日付を動的に無効にします。

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## 過去/未来の無効化

`disablePast` または `disableFuture` を使用して、今日より前の日付または今日より後の日付をすばやく無効にします。

<CodeDemo source={DisablePastFutureSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <DisablePastFuture client:load />
  </ExampleWrapper>
</CodeDemo>

## 日付のハイライト

`highlightDates` propを使用して、特定の日付（例：休日、イベント）を視覚的にハイライトします。

<CodeDemo source={HighlightDatesSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <HighlightDates client:load />
  </ExampleWrapper>
</CodeDemo>
