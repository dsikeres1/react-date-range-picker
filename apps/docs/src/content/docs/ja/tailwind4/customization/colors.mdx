# カラー

ラッパー要素の CSS カスタムプロパティを上書きすることで、実行時にカラーテーマを切り替えることができます。Tailwind v4 パッケージはすべてのカラーをセマンティックトークンから読み取るため、基になる CSS 変数を変更すると、ピッカー全体が即座にリブランドされます。

Tailwind v4 パッケージは、セマンティックトークンに **oklch** フォーマットを使用します。`--color-primary` と `--color-primary-foreground` を上書きしてピッカーをリブランドします:

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:load>
    <CompoundColorTheme client:load />
  </ExampleWrapper>
</CodeDemo>
