# タイポグラフィ

Tailwind v4 パッケージは、Tailwind のデフォルトの `font-sans` クラス (`ui-sans-serif, system-ui, sans-serif, ...`) を使用します。CSS の `font-family` は継承されるため、ピッカーを `fontFamily` スタイルを持つコンテナでラップすることで、フォントを上書きできます。ラッパーのフォントは、すべてのピッカー要素に継承されます。

## フォントファミリー

ピッカーをラップし、ラッパーに `fontFamily` を設定してフォントを切り替えます:

<CodeDemo source={CompoundCustomFontSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundCustomFont client:visible />
  </ExampleWrapper>
</CodeDemo>

## Webフォントの使用 (Noto Sans)

実行時にWebフォントを読み込み、ラッパーを介して適用します。この例では、Google Fontsから[Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans)を読み込みます:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="本番環境での使用">
実際のプロジェクトでは、実行時にフォントを注入するのではなく、HTMLの `<head>` 内の `<link>` タグを使用するか、CSSバンドラを介してフォントを読み込んでください。ここでの `useEffect` を使用したアプローチは、デモンストレーションのみを目的としています。
</Admonition>
