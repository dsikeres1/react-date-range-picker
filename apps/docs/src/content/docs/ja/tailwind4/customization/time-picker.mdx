# タイムピッカー

`TimePicker`コンパウンドコンポーネントを使用して、カスタムの時間選択UIを構築します。この例では、デフォルトのトリガーをスタイル付きのボタンに置き換えつつ、標準のタイムパネルとフッターは維持しています。
