# カスタムフッター

## フッターの再配置

フッターボタンを再配置したり、`TodayButton`のような新しいボタンを追加したりできます。各ボタンは独立した複合パーツであるため、順序やレイアウトを自由に制御できます。

    <CompoundCustomFooter client:visible />
  </ExampleWrapper>
</CodeDemo>

## 最小レイアウト

不要なパーツは省略できます。例えば、フッターを完全に取り除いて、最小限のポップアップカレンダーを作成できます：

</CodeDemo>

## クラスの上書き

すべての複合パーツは `className` prop を受け入れます。The Tailwind v4 パッケージは `cn()` (a `clsx` + `twMerge` ユーティリティ) を使用して、あなたのクラスをデフォルトとマージするため、競合するユーティリティは正しく解決されます：

```tsx
import { DatePicker } from "react-date-range-picker-tailwind4";

<DatePicker.Root value={value} onChange={setValue}>
  {/* Override trigger width and padding */}
  <DatePicker.Trigger className="min-w-[300px] px-4 py-3" />
  <DatePicker.Content className="rounded-2xl shadow-xl">
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title className="text-lg" />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>;
```
