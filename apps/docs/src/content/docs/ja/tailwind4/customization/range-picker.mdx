# レンジピッカー

複合コンポーネントを使用して、完全にカスタマイズされた `DateRangePicker` を構築します。この例では、カスタムトリガー、プリセットサイドバー、デュアルカレンダーグリッド、フッターを組み合わせています。これらはすべて個別のパーツから構成されています。
