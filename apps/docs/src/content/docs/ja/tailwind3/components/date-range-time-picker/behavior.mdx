# 挙動

## 選択時に閉じる

`shouldCloseOnSelect`が有効な場合、プリセットをクリックするとピッカーが自動的に確定して閉じます。

<Admonition type="info">
  時間入力が有効なピッカーの場合、`shouldCloseOnSelect`はプリセットのクリックにのみ適用されます。ユーザーが時間を設定する必要があるため、日付のクリックには適用されません。
</Admonition>

  </ExampleWrapper>
</CodeDemo>
