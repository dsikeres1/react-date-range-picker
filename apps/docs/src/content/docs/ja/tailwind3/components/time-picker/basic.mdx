# TimePicker

入力トリガーとポップアップパネルを備えたタイムピッカー。入力をクリックすると、スクロール可能な時間列を持つドロップダウンが開きます。

## 基本的な使い方

<CodeDemo source={BasicSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <Basic client:visible />
  </ExampleWrapper>
</CodeDemo>

## インライン

トリガーやポップアップなしで、時間パネルをインラインでレンダリングします。

<CodeDemo source={InlineSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <Inline client:visible />
  </ExampleWrapper>
</CodeDemo>
