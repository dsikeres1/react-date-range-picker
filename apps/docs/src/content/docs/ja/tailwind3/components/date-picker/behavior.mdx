# 動作

## 選択時に閉じる

`shouldCloseOnSelect`が有効な場合、日付をクリックするとピッカーが自動的に確定して閉じます。

    <ShouldCloseOnSelect client:load />
  </ExampleWrapper>
</CodeDemo>

## 複数月の表示

`name` propを使用して、ネイティブのフォーム送信用の非表示inputを含めることができます。

  この例の送信ボタンのスタイルはカスタムです — ライブラリが提供するのは日付ピッカーのみです。
</Admonition>
