# 時間オプション

`time` prop を使用して、時間パネルの動作を設定します。

## 12時間形式

AM/PMトグル付きの12時間形式に切り替えます。

<CodeDemo source={TwelveHourSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <TwelveHour client:visible />
  </ExampleWrapper>
</CodeDemo>

## 分単位のステップ

分の増分ステップを設定します。固定間隔でのスケジューリングに便利です。

<CodeDemo source={MinuteStepSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinuteStep client:visible />
  </ExampleWrapper>
</CodeDemo>

## 秒単位の精度

`precision` を `"second"` に設定すると、秒の列が表示されます。
