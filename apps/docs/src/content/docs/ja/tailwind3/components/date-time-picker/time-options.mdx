# 時間オプション

`time` prop を使用して、時間パネルの動作を設定します。

## 分のステップ

分の増分ステップを設定します。固定間隔でのスケジューリングに便利です。

<CodeDemo source={MinuteStepSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinuteStep client:visible />
  </ExampleWrapper>
</CodeDemo>

## 秒の精度

`precision` を `"second"` に設定して、秒の列を表示します。

<CodeDemo source={SecondPrecisionSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <SecondPrecision client:visible />
  </ExampleWrapper>
</CodeDemo>

## 12時間形式

AM/PM トグル付きの12時間形式に切り替えます。
