# 動作

## 選択時に閉じる

`shouldCloseOnSelect` が有効な場合、終了日が選択されるとピッカーは自動的に確定し、閉じます。

<CodeDemo source={ShouldCloseOnSelectSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <ShouldCloseOnSelect client:load />
  </ExampleWrapper>
</CodeDemo>

## 複数月表示

デフォルトの2ヶ月以上を並べて表示します。

<CodeDemo source={MultiMonthSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <MultiMonth client:load />
  </ExampleWrapper>
</CodeDemo>

## フォーム連携

`name` prop を使用して、ネイティブフォームの送信用の非表示 input を含めることができます。

  この例の送信ボタンのスタイルはカスタムです — ライブラリが提供するのは日付ピッカーのみです。
</Admonition>
