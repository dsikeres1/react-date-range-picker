# タイポグラフィ

Tailwind v3パッケージは、Tailwindのデフォルトの`font-sans`クラス（`ui-sans-serif, system-ui, sans-serif, ...`）を使用します。CSSの`font-family`は継承されるため、ピッカーを`fontFamily`スタイルを持つコンテナでラップすることでフォントを上書きできます。ラッパーのフォントは、すべてのピッカー要素に継承されます。

## フォントファミリー

ピッカーをラップし、ラッパーに`fontFamily`を設定してフォントを切り替えます:

<CodeDemo source={CompoundCustomFontSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundCustomFont client:visible />
  </ExampleWrapper>
</CodeDemo>

## Webフォントの使用 (Noto Sans)

実行時にWebフォントを読み込み、ラッパーを介して適用します。この例では、Google Fontsから[Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans)を読み込みます:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="本番環境での使用">
実際のプロジェクトでは、実行時にフォントを注入する代わりに、HTMLの`<head>`内の`<link>`タグを介して、またはCSSバンドラを通じてフォントを読み込みます。ここでの`useEffect`アプローチは、デモンストレーションのみを目的としています。
</Admonition>
