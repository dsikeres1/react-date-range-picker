# カスタムフッター

## フッターの再配置

フッターボタンを再配置し、「今日」ボタンのような追加アクションを追加します。複合コンポーネントを使用すると、どのボタンがどの順序で表示されるかを正確に制御できます。

## クラスの上書き

すべての複合パーツは `className` プロパティを受け入れます。Tailwind v3 パッケージは `cn()` (`clsx` と `twMerge` のユーティリティ) を使用して、あなたのクラスをデフォルトとマージするため、競合するユーティリティが正しく解決されます:

```tsx
import { DatePicker } from "react-date-range-picker-tailwind3";

<DatePicker.Root value={value} onChange={setValue}>
  <DatePicker.Trigger className="min-w-[300px] px-4 py-3" />
  <DatePicker.Content className="rounded-2xl shadow-xl">
    <DatePicker.Header>
      <DatePicker.PrevButton />
      <DatePicker.Title className="text-lg" />
      <DatePicker.NextButton />
    </DatePicker.Header>
    <DatePicker.Grid />
  </DatePicker.Content>
</DatePicker.Root>;
```
