# カラー

Tailwind v3パッケージは、CSS変数の代わりに**ハードコードされたTailwindユーティリティクラス**（例: `bg-sky-500`）を使用します。色を切り替えるには、Compound APIを使用して日のクラス名を動的にオーバーライドします:

<Admonition type="info">
  styledパッケージ（CSS変数を使用）とは異なり、Tailwind
  v3は実行時に色をオーバーライドするためにCompound
  APIが必要です。詳細については、[テーマのオーバーライド](/docs/tailwind3/customization/theme-override)を参照してください。
</Admonition>
