# Farben

Das Tailwind v3-Paket verwendet **festcodierte Tailwind-Hilfsklassen** (z. B. `bg-sky-500`) anstelle von CSS-Variablen. Um Farben zu wechseln, verwenden Sie die Compound-API und überschreiben Sie die Tages-Klassennamen dynamisch:

<Admonition type="info">
  Im Gegensatz zum Styled-Paket (das CSS-Variablen verwendet) erfordert Tailwind v3 die
  Compound-API, um Farben zur Laufzeit zu überschreiben. Weitere Details finden Sie unter [Theme
  Override](/docs/tailwind3/customization/theme-override).
</Admonition>
