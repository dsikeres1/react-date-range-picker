# Benutzerdefinierte Tageszelle

Verwenden Sie die Render-Prop der `Day`-Komponente, um jede Tageszelle anzupassen. Dies ist nützlich, um Ereignisanzeigen hinzuzufügen, benutzerdefinierte Stile basierend auf Datumseigenschaften anzuwenden oder das Rendern des Tages vollständig zu ersetzen.
