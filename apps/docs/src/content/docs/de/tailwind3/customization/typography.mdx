# Typografie

Das Tailwind v3-Paket verwendet die standardmäßige `font-sans`-Klasse von Tailwind (`ui-sans-serif, system-ui, sans-serif, ...`). Da die CSS-Eigenschaft `font-family` vererbt wird, können Sie die Schriftart des Pickers überschreiben, indem Sie ihn in einen Container mit einem `fontFamily`-Stil einbetten. Die Schriftart des Wrappers wird auf alle Elemente des Pickers kaskadiert.

## Schriftfamilie

Umschließen Sie den Picker und setzen Sie `fontFamily` auf den Wrapper, um die Schriftarten zu wechseln:

## Verwendung von Web-Schriftarten (Noto Sans)

Laden Sie zur Laufzeit eine Web-Schriftart und wenden Sie sie über einen Wrapper an. Dieses Beispiel lädt [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) von Google Fonts:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <CompoundNotoSans client:load />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="Produktionseinsatz">
In einem echten Projekt laden Sie die Schriftart über ein `<link>`-Tag in Ihrem HTML-`<head>` oder über Ihren CSS-Bundler, anstatt sie zur Laufzeit zu injizieren. Der hier gezeigte `useEffect`-Ansatz dient nur zu Demonstrationszwecken.
</Admonition>
