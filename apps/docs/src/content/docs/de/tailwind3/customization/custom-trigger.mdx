# Eigener Trigger

Ersetzen Sie den Standard-Trigger durch Ihr eigenes Element, indem Sie eine Render-Prop verwenden. Die `Trigger`-Komponente akzeptiert eine Funktion als Kind, die Zustand und Callbacks empfängt, sodass Sie jeden beliebigen benutzerdefinierten Button oder Input rendern können.

  </ExampleWrapper>
</CodeDemo>

## Trigger-Render-Props

| Prop           | Typ              | Beschreibung                    |
| -------------- | ---------------- | ------------------------------- |
| `displayValue` | `string`         | Formatiertes ausgewähltes Datum |
| `placeholder`  | `string`         | Platzhaltertext                 |
| `hasValue`     | `boolean`        | Ob ein Datum ausgewählt ist     |
| `isOpen`       | `boolean`        | Ob das Popup geöffnet ist       |
| `onToggle`     | `() => void`     | Popup öffnen/schließen          |
| `onClear`      | `() => void`     | Auswahl löschen                 |
| `locale`       | `Locale`         | Aktuelles Locale-Objekt         |
| `triggerRef`   | `(node) => void` | Ref-Callback zur Positionierung |
