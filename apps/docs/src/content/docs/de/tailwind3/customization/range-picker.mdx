# Bereichsauswahl

Die `DateRangePicker` Compound-API bietet zusätzliche Teile für duale Kalender und vordefinierte Bereiche. Sie erhalten die volle Kontrolle über das Layout von Voreinstellungen, Kalenderrastern, Kopfzeilen und Fußzeilen-Aktionen.
