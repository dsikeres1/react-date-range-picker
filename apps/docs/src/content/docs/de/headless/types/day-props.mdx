# DayProps

Pro Zelle berechnete Flags, die von `getDayProps()` zurückgegeben werden. Beschreibt den visuellen und interaktiven Zustand einer einzelnen Kalendertag-Zelle.

```tsx
import type { DayProps } from "react-date-range-picker-headless";
```

## Felder

| Feld   | Typ    | Beschreibung                     |
| ------ | ------ | -------------------------------- |
| `date` | `Date` | Das Date-Objekt für diese Zelle. |
| `day`  |
