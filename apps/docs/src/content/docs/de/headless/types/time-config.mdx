# TimeConfig

Konfigurationsobjekt für zeitbezogene Auswahlkomponenten. Wird von `useDateTimePicker`, `useDateRangeTimePicker` und `useStandaloneTimePicker` gemeinsam genutzt.

```tsx
import type { TimeConfig } from "react-date-range-picker-headless";
```
