# Benutzerdefinierter Auslöser

Ersetzen Sie den Standardauslöser durch Ihren eigenen Button oder Ihre eigene Eingabe. Die `Trigger`-Komponente akzeptiert eine Render-Prop, die Ihnen die volle Kontrolle über das Aussehen und Verhalten des Auslösers gibt.

    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Trigger Render-Props

| Prop           | Typ              | Beschreibung                      |
| -------------- | ---------------- | --------------------------------- |
| `displayValue` | `string`         | Formatiertes ausgewähltes Datum   |
| `placeholder`  | `string`         | Platzhaltertext                   |
| `hasValue`     | `boolean`        | Ob ein Datum ausgewählt ist       |
| `isOpen`       | `boolean`        | Ob das Popup geöffnet ist         |
| `onToggle`     | `() => void`     | Popup öffnen/schließen umschalten |
| `onClear`      | `() => void`     | Die Auswahl löschen               |
| `locale`       | `Locale`         | Aktuelles Locale-Objekt           |
| `triggerRef`   | `(node) => void` | Ref-Callback zur Positionierung   |
