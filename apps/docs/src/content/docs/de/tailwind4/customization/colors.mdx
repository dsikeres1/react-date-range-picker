# Farben

Wechseln Sie Farbschemata zur Laufzeit, indem Sie CSS-benutzerdefinierte Eigenschaften auf einem Wrapper-Element überschreiben. Da das Tailwind v4-Paket alle Farben aus semantischen Tokens liest, wird durch Ändern der zugrunde liegenden CSS-Variablen der gesamte Picker sofort neu gebrandet.

Das Tailwind v4-Paket verwendet das **oklch**-Format für semantische Tokens. Überschreiben Sie `--color-primary` und `--color-primary-foreground`, um den Picker neu zu branden:

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>
