# Zeitauswahl

Verwenden Sie die `TimePicker`-Verbundkomponenten, um eine benutzerdefinierte Benutzeroberfläche für die Zeitauswahl zu erstellen. Dieses Beispiel ersetzt den Standard-Trigger durch eine gestaltete Schaltfläche, während das Standard-Zeitpanel und die Fußzeile beibehalten werden.
