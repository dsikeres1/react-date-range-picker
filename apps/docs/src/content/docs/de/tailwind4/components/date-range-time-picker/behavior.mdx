# Verhalten

## Schließen bei Auswahl

Wenn `shouldCloseOnSelect` aktiviert ist, bestätigt und schließt der Picker automatisch, wenn eine Voreinstellung angeklickt wird.

<Admonition type="info">
  Bei Pickern mit Zeitauswahl gilt `shouldCloseOnSelect` nur für Klicks auf Voreinstellungen – nicht
  für Klicks auf Daten, da der Benutzer noch die Uhrzeit einstellen muss.
</Admonition>
