# Einschränkungen

## Erforderlich

Wenn `required` gesetzt ist, ist die Löschen-Schaltfläche deaktiviert und der Wert kann nicht geleert werden.

<CodeDemo source={RequiredSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <Required client:visible />
  </ExampleWrapper>
</CodeDemo>

## Mindest- & Höchstdatum

Beschränken Sie auswählbare Daten auf einen bestimmten Bereich mit `minDate` und `maxDate`.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <MinMaxDate client:visible />
  </ExampleWrapper>
</CodeDemo>

## Deaktivierte Daten

Verwenden Sie `isDateUnavailable`, um bestimmte Daten dynamisch zu deaktivieren.

<CodeDemo source={DisabledDatesSource}>
