# Colors

The Tailwind v3 package uses **hardcoded Tailwind utility classes** (e.g., `bg-sky-500`) instead of CSS variables. To switch colors, use the Compound API and override day class names dynamically:

<Admonition type="info">
  Unlike the styled package (which uses CSS variables), Tailwind v3 requires the Compound API to
  override colors at runtime. See [Theme Override](/docs/tailwind3/customization/theme-override) for
  more details.
</Admonition>
