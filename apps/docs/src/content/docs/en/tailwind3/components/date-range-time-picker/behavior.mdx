# Behavior

## Close on Select

When `shouldCloseOnSelect` is enabled, the picker automatically confirms and closes when a preset is clicked.

<Admonition type="info">
  For time-enabled pickers, `shouldCloseOnSelect` only applies to preset clicks — not date clicks,
  since the user still needs to set the time.
</Admonition>
