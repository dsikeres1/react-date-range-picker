# Typography

The styled package uses a **system font stack** by default:

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

This renders with the OS native font on each platform (San Francisco on macOS, Segoe UI on Windows, Roboto on Android). System fonts provide good multi-language support since the OS handles CJK, Arabic, and other script fallbacks automatically.

To override, set `--rdrp-font-family` on any ancestor element — it cascades into the picker.

## Using Web Fonts (Noto Sans)

For consistent cross-platform rendering or i18n projects, load a web font like [Noto Sans](https://fonts.google.com/noto) and override the variable:

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/rdrp-styles.css";

// 1. Load Noto Sans in your HTML or CSS:
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. Override the font variable:
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## Font Size & Weight

Fine-tune typography density by overriding size and weight variables:

| Variable                      | Default | Description           |
| ----------------------------- | ------- | --------------------- |
| `--rdrp-font-size-xs`         | `11px`  | Smallest text         |
| `--rdrp-font-size-sm`         | `12px`  | Weekday labels        |
| `--rdrp-font-size-base`       | `13px`  | Day cells, buttons    |
| `--rdrp-font-size-md`         | `14px`  | Trigger, time items   |
| `--rdrp-font-size-lg`         | `15px`  | Month title           |
| `--rdrp-font-weight-normal`   | `400`   | Base weight           |
| `--rdrp-font-weight-medium`   | `500`   | Medium emphasis       |
| `--rdrp-font-weight-semibold` | `600`   | Selected days, titles |
| `--rdrp-font-weight-bold`     | `700`   | Today indicator       |
