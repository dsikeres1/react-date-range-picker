# Colors

Switch color themes at runtime by overriding CSS custom properties on a wrapper element. Since the Tailwind v4 package reads all colors from semantic tokens, changing the underlying CSS variables instantly rebrands the entire picker.

The Tailwind v4 package uses **oklch** format for semantic tokens. Override `--color-primary` and `--color-primary-foreground` to rebrand the picker:

</CodeDemo>
