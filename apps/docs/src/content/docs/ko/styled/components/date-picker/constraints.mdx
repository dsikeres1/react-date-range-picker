# 제약조건

## 필수값

`required`가 설정되면 지우기 버튼이 비활성화되며 값을 비울 수 없습니다.

    <Required client:load />
  </ExampleWrapper>
</CodeDemo>

## 최소 & 최대 날짜

`minDate`와 `maxDate`를 사용하여 선택 가능한 날짜를 특정 범위로 제한합니다.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="styled" client:load>
    <MinMaxDate client:load />
  </ExampleWrapper>
</CodeDemo>

`isDateUnavailable`을 사용하여 특정 날짜를 동적으로 비활성화합니다.

`disablePast` 또는 `disableFuture`를 사용하여 오늘 이전/이후의 날짜를 쉽게 비활성화할 수 있습니다.
