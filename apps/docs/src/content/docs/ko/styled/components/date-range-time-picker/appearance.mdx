## 다중 달력

2개 이상의 달력을 나란히 표시합니다. 범위 선택기(range picker)의 기본값은 `2`입니다.

헤드리스 패키지의 `createLocale`을 사용하여 월/일 이름과 UI 문자열을 사용자 지정할 수 있습니다.
