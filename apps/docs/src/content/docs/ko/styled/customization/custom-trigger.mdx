# 커스텀 트리거

기본 트리거를 자체 버튼이나 입력 필드로 교체하세요. `Trigger` 컴포넌트는 트리거의 모양과 동작을 완벽하게 제어할 수 있는 렌더 프롭(render prop)을 받습니다.

  </ExampleWrapper>

## 트리거 렌더 프롭

| 속성           | 타입             | 설명                      |
| -------------- | ---------------- | ------------------------- |
| `displayValue` | `string`         | 포맷팅된 선택 날짜        |
| `placeholder`  | `string`         | 플레이스홀더 텍스트       |
| `hasValue`     | `boolean`        | 날짜 선택 여부            |
| `isOpen`       | `boolean`        | 팝업 열림 여부            |
| `onToggle`     | `() => void`     | 팝업 열기/닫기 토글       |
| `onClear`      | `() => void`     | 선택 초기화               |
| `locale`       | `Locale`         | 현재 로케일 객체          |
| `triggerRef`   | `(node) => void` | 위치 지정을 위한 Ref 콜백 |
