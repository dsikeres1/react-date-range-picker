# 색상

래퍼 요소에서 CSS 사용자 지정 속성(custom properties)을 재정의하여 런타임에 색상 테마를 전환하세요. Tailwind v4 패키지는 시맨틱 토큰에서 모든 색상을 읽으므로, 기본 CSS 변수를 변경하면 전체 피커의 브랜드 색상이 즉시 변경됩니다.

Tailwind v4 패키지는 시맨틱 토큰에 **oklch** 형식을 사용합니다. `--color-primary`와 `--color-primary-foreground`를 재정의하여 피커의 브랜드를 변경하세요:

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>
