# 동작

## 선택 시 닫기

`shouldCloseOnSelect`가 활성화되면 프리셋을 클릭할 때 피커가 자동으로 확인되고 닫힙니다.

<Admonition type="info">
  시간 기능이 있는 피커의 경우, 사용자가 여전히 시간을 설정해야 하므로 `shouldCloseOnSelect`는 날짜
  클릭이 아닌 프리셋 클릭에만 적용됩니다.
</Admonition>
