# 제약 조건

## 필수

`required`가 설정되면 지우기 버튼이 비활성화되고 값을 비울 수 없습니다.

<CodeDemo source={RequiredSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <Required client:visible />
  </ExampleWrapper>
</CodeDemo>

## 최소 및 최대 날짜

`minDate` 및 `maxDate`를 사용하여 선택 가능한 날짜를 특정 범위로 제한합니다.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinMaxDate client:visible />
  </ExampleWrapper>
</CodeDemo>

## 비활성화된 날짜

`isDateUnavailable`을 사용하여 특정 날짜를 동적으로 비활성화합니다.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <DisabledDates client:visible />
  </ExampleWrapper>
</CodeDemo>

## 과거 / 미래 비활성화

`disablePast` 또는 `disableFuture`를 사용하여 오늘 이전/이후의 날짜를 빠르게 비활성화합니다.

<CodeDemo source={DisablePastFutureSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <DisablePastFuture client:visible />
  </ExampleWrapper>
</CodeDemo>

## 날짜 강조 표시

`highlightDates` prop을 사용하여 특정 날짜(예: 공휴일, 이벤트)를 시각적으로 강조 표시합니다.

<CodeDemo source={HighlightDatesSource}>

  </ExampleWrapper>
</CodeDemo>
