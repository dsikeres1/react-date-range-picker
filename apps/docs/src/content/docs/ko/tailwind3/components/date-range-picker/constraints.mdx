# 제약 조건

## 최소 및 최대 일수

`minDays`와 `maxDays`를 사용하여 범위의 길이를 제한할 수 있습니다(시작일과 종료일 포함). 사용자는 `minDays`보다 짧거나 `maxDays`보다 긴 범위를 확정할 수 없습니다.

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <MinMaxDays client:load />
  </ExampleWrapper>
</CodeDemo>

## 선택 불가 날짜

`isDateUnavailable`을 사용하여 특정 날짜를 동적으로 비활성화할 수 있습니다. 비활성화된 날짜는 범위의 시작이나 끝으로 선택할 수 없습니다.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind3" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## 단일 날짜 허용

기본적으로 시작일과 종료일이 같은 범위가 허용됩니다. 최소 두 개의 다른 날짜를 요구하려면 `allowSingleDateInRange={false}`로 설정하세요.
