# 커스텀 트리거

렌더 프롭(render prop)을 사용하여 기본 트리거를 커스텀 엘리먼트로 교체하세요. `Trigger` 컴포넌트는 상태와 콜백을 받는 함수를 자식으로 허용하여 어떤 형태의 커스텀 버튼이나 입력 필드라도 렌더링할 수 있게 해줍니다.

## 트리거 렌더 프롭

| Prop           | Type             | Description               |
| -------------- | ---------------- | ------------------------- |
| `displayValue` | `string`         | 포맷팅된 선택 날짜        |
| `placeholder`  | `string`         | 플레이스홀더 텍스트       |
| `hasValue`     | `boolean`        | 날짜가 선택되었는지 여부  |
| `isOpen`       | `boolean`        | 팝업이 열려있는지 여부    |
| `onToggle`     | `() => void`     | 팝업 열기/닫기 토글       |
| `onClear`      | `() => void`     | 선택 초기화               |
| `locale`       | `Locale`         | 현재 로케일 객체          |
| `triggerRef`   | `(node) => void` | 위치 지정을 위한 ref 콜백 |
