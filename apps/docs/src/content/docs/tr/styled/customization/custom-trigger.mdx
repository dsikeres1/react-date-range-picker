# Özel Tetikleyici

Varsayılan tetikleyiciyi kendi düğmeniz veya girdinizle değiştirin. `Trigger` bileşeni, tetikleyicinin görünümü ve davranışı üzerinde tam kontrol sağlayan bir render prop kabul eder.

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Tetikleyici Render Props

| Prop           | Tür              | Açıklama                             |
| -------------- | ---------------- | ------------------------------------ |
| `displayValue` | `string`         | Biçimlendirilmiş seçili tarih        |
| `placeholder`  | `string`         | Yer tutucu metin                     |
| `hasValue`     | `boolean`        | Bir tarihin seçili olup olmadığı     |
| `isOpen`       | `boolean`        | Açılır pencerenin açık olup olmadığı |
| `onToggle`     | `() => void`     | Açılır pencereyi aç/kapat            |
| `onClear`      | `() => void`     | Seçimi temizle                       |
| `locale`       | `Locale`         | Mevcut yerel ayar nesnesi            |
| `triggerRef`   | `(node) => void` | Konumlandırma için ref geri araması  |
