# Boyutlar

`styled` paketi, tipografi, boşluk, hücre boyutları ve tetikleyici boyutlarını orantılı olarak ölçeklendiren 4 yerleşik boyut varyantı içerir.

## Mevcut Boyutlar

| Boyut     | Hücre Boyutu | Yazı Tipi Tabanı | Tetikleyici Min Genişliği |
| --------- | ------------ | ---------------- | ------------------------- |
| `small`   | 28px         | 11px             | 160px                     |
| `medium`  | 36px         | 13px             | 200px                     |
| `large`   | 44px         | 14px             | 240px                     |
| `x-large` | 52px         | 15px             | 280px                     |

`medium` varsayılandır.

## Kullanım

`size` prop'unu herhangi bir seçici bileşenine geçirin:

<CodeDemo source={SizesSource}>
  <ExampleWrapper variant="styled" client:visible>
    <Sizes client:visible />
  </ExampleWrapper>
</CodeDemo>

## Özel Boyutlar

Boyutlandırma değişkenlerini geçersiz kılan bir CSS sınıfı tanımlayarak kendi boyutunuzu oluşturun:

```css
.rdrp-size-custom {
  --rdrp-cell-size: 40px;
  --rdrp-nav-btn-size: 30px;
  --rdrp-font-size-base: 14px;
  --rdrp-font-size-md: 15px;
  --rdrp-trigger-min-width: 220px;
  --rdrp-popup-padding: 18px;
}
```

Ardından bir sarmalayıcı aracılığıyla uygulayın:

```tsx
<div className="rdrp-size-custom">
  <DatePicker value={value} onChange={setValue} />
</div>
```

Veya Bileşik Bileşen API'si ile sınıfı Root'a ekleyin:

```tsx
<DatePicker.Root value={value} onChange={setValue} className="rdrp-size-custom">
  {/* ... */}
</DatePicker.Root>
```

Ayrıca satır içi stiller aracılığıyla boyutlandırma değişkenlerini geçersiz kılabilirsiniz:
