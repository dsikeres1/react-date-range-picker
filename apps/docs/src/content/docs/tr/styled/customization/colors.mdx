# Renkler

Bir sarmalayıcı (wrapper) element üzerindeki CSS özel özelliklerini (custom properties) geçersiz kılarak çalışma zamanında renk temasını değiştirin. Stilli paket, tüm birincil vurgu renkleri için `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light` ve `--rdrp-color-primary-lighter` kullanır.

## Satır-içi Tema Değiştirici

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="styled" client:load>
    <CompoundColorTheme client:load />
  </ExampleWrapper>
</CodeDemo>

## Açılır Pencere Tema Değiştirici

Aynı yaklaşım açılır pencere seçicilerle de çalışır:

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="styled" client:load>
    <CompoundColorThemePopup client:load />
  </ExampleWrapper>
</CodeDemo>

## CSS Değişkeni Satır-içi Stili

Ayrıca bir sarmalayıcı (wrapper) element üzerindeki `style` niteliği aracılığıyla renkleri doğrudan geçersiz kılabilirsiniz:

<CodeDemo source={CSSCustomColorsSource}>
  <ExampleWrapper variant="styled" client:load>
    <CSSCustomColors client:load />
  </ExampleWrapper>
</CodeDemo>
