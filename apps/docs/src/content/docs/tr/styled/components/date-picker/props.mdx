# Prop'lar

| Prop                  | Tür                                           | Varsayılan  | Açıklama                                                |
| --------------------- | --------------------------------------------- | ----------- | ------------------------------------------------------- |
| `value`               | `Date \| null`                                | --          | Seçili tarih değeri                                     |
| `onChange`            | `(date: Date \| null) => void`                | --          | Değer değiştiğinde çağrılan geri arama fonksiyonu       |
| `minDate`             | `Date`                                        | --          | Seçilebilecek minimum tarih                             |
| `maxDate`             | `Date`                                        | --          | Seçilebilecek maksimum tarih                            |
| `locale`              | `Partial<Locale>`                             | --          | Özel yerelleştirme yapılandırması                       |
| `initialMonth`        | `Date`                                        | --          | Başlangıçta görüntülenen ay                             |
| `size`                | `"small" \| "medium" \| "large" \| "x-large"` | `"medium"`  | Bileşen boyutu                                          |
| `weekStartsOn`        | `WeekDay`                                     | `"sunday"`  | Haftanın ilk günü                                       |
| `isDateUnavailable`   | `(date: Date) => boolean`                     | --          | Tarihleri devre dışı olarak işaretle                    |
| `displayFormat`       | `string`                                      | --          | Özel görüntüleme formatı (ör. `"YYYY/MM/DD"`)           |
| `open`                | `boolean`                                     | --          | Kontrollü açık durumu                                   |
| `initialOpen`         | `boolean`                                     | --          | Başlangıçtaki açık durumu (kontrolsüz)                  |
| `onOpenChange`        | `(open: boolean) => void`                     | --          | Açık durumu değiştiğinde çağrılan geri arama fonksiyonu |
| `required`            | `boolean`                                     | `false`     | Değerin temizlenmesini devre dışı bırak                 |
| `showOutsideDays`     | `boolean`                                     | `false`     | Bitişik aylardaki günleri göster                        |
| `highlightDates`      | `Date[]`                                      | --          | Vurgulanacak tarihler                                   |
| `shouldCloseOnSelect` | `boolean`                                     | `false`     | Tarihe tıklandığında otomatik onayla                    |
| `numberOfMonths`      | `number`                                      | `1`         | Görüntülenecek ay sayısı                                |
| `captionLayout`       | `"buttons" \| "dropdown"`                     | `"buttons"` | Başlık navigasyon stili                                 |
| `fromYear`            | `number`                                      | --          | Açılır menü için başlangıç yılı                         |
| `toYear`              | `number`                                      | --          | Açılır menü için bitiş yılı                             |
| `disablePast`         | `boolean`                                     | `false`     | Bugünden önceki tarihleri devre dışı bırak              |
| `disableFuture`       | `boolean`                                     | `false`     | Bugünden sonraki tarihleri devre dışı bırak             |
| `inline`              | `boolean`                                     | `false`     | Takvimi satır içi göster (açılır pencere olmadan)       |
| `name`                | `string`                                      | --          | Form gönderimi için gizli input adı                     |
| `placeholder`         | `string`                                      | --          | Tetikleyici yer tutucu metni                            |
| `hideHeader`          | `boolean`                                     | `false`     | Navigasyon başlığını gizle                              |
| `hideFooter`          | `boolean`                                     | `false`     | Eylem altbilgisini gizle                                |
