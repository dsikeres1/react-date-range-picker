# Renkler

Bir sarmalayıcı (wrapper) element üzerindeki CSS özel özelliklerini (custom properties) geçersiz kılarak çalışma zamanında renk temalarını değiştirin. Tailwind v4 paketi tüm renkleri semantik token'lardan okuduğu için, altta yatan CSS değişkenlerini değiştirmek tüm seçiciyi anında yeniden markalar.

Tailwind v4 paketi, semantik token'lar için **oklch** formatını kullanır. Seçiciyi yeniden markalamak için `--color-primary` ve `--color-primary-foreground` değişkenlerini geçersiz kılın:

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>
