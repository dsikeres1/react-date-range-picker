# Renkler

Tailwind v3 paketi, CSS değişkenleri yerine **sabit kodlanmış Tailwind yardımcı sınıflarını** (ör. `bg-sky-500`) kullanır. Renkleri değiştirmek için Bileşik (Compound) API'sini kullanın ve gün sınıf adlarını dinamik olarak geçersiz kılın:

<Admonition type="info">
  Styled paketinin (CSS değişkenleri kullanan) aksine, Tailwind v3 çalışma zamanında renkleri
  geçersiz kılmak için Bileşik (Compound) API'sini gerektirir. Daha fazla ayrıntı için [Tema
  Geçersiz Kılma](/docs/tailwind3/customization/theme-override) bölümüne bakın.
</Admonition>
