# Proplar

| Prop            | Tür                            | Varsayılan | Açıklama                                               |
| --------------- | ------------------------------ | ---------- | ------------------------------------------------------ |
| `value`         | `Date \| null`                 | --         | Seçilen zaman değeri (yalnızca zaman kısmı kullanılır) |
| `onChange`      | `(date: Date \| null) => void` | --         | Değer değiştiğinde geri arama                          |
| `time`          | `TimeConfig`                   | --         | Zaman paneli yapılandırması                            |
| `locale`        | `Partial<Locale>`              | --         | Özel yerel ayar yapılandırması                         |
| `displayFormat` | `string`                       | --         | Tetikleyici için özel görüntüleme formatı              |
| `placeholder`   | `string`                       | --         | Tetikleyici için yer tutucu metni                      |
| `open`          | `boolean`                      | --         | Kontrollü açık durumu                                  |
| `initialOpen`   | `boolean`                      | `false`    | Başlangıçtaki açık durumu (kontrolsüz)                 |
| `onOpenChange`  | `(open: boolean) => void`      | --         | Açık durumu değiştiğinde geri arama                    |
| `required`      | `boolean`                      | `false`    | Değerin temizlenmesini devre dışı bırak                |
| `inline`        | `boolean`                      | `false`    | Tetikleyici/açılır pencere olmadan satır içi oluştur   |
| `hideFooter`    | `boolean`                      | --         | Eylem altbilgisini gizle                               |
| `name`          | `string`                       | --         | Form gönderimi için gizli girdi adı                    |

### TimeConfig

```ts
interface TimeConfig {
  /** Zaman görüntüleme hassasiyeti. Varsayılan "minute". */
  precision?: "hour" | "minute" | "second";
  /** 12 saatlik veya 24 saatlik format. Varsayılan "24". */
  hourFormat?: "12" | "24";
  /** Dakika artış adımı. Varsayılan 5. */
  minuteStep?: 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
  /** Saniye artış adımı. Varsayılan 1. */
  secondStep?: 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
  /** Her bir kaydırma tekerleği öğesinin yüksekliği (piksel cinsinden). Varsayılan 32. */
  itemHeight?: number;
}
```
