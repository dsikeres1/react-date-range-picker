# 顏色

`Tailwind v3` 套件使用**寫死的 Tailwind 功能類別**（例如 `bg-sky-500`），而非 CSS 變數。若要切換顏色，請使用 Compound API 並動態覆寫日期類別名稱：

<Admonition type="info">
  與使用 CSS 變數的 `styled` 套件不同，`Tailwind v3` 需要使用 Compound API
  在執行時覆寫顏色。詳情請參閱 [主題覆寫](/docs/tailwind3/customization/theme-override)。
</Admonition>
