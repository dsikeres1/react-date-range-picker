# 行為

## 選擇後關閉

當啟用 `shouldCloseOnSelect` 時，點擊預設選項後，選擇器會自動確認並關閉。

<Admonition type="info">
  對於啟用時間功能的選擇器，`shouldCloseOnSelect` 僅適用於點擊預設選項 —
  而非點擊日期，因為使用者仍需設定時間。
</Admonition>
