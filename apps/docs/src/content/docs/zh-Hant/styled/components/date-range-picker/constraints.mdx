# 限制

## 最小與最大天數

使用 `minDays` 和 `maxDays` (包含) 來限制範圍長度。使用者無法確認短於 `minDays` 或長於 `maxDays` 的範圍。

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="styled" client:load>
    <MinMaxDays client:load />
  </ExampleWrapper>
</CodeDemo>

## 禁用日期

使用 `isDateUnavailable` 來動態禁用特定日期。被禁用的日期不能被選為範圍的開始或結束。

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="styled" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## 允許單一日期

預設情況下，允許開始與結束日期相同的範圍。設定 `allowSingleDateInRange={false}` 來要求至少兩個不同的日期。
