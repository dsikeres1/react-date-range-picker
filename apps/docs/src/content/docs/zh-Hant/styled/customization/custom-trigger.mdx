# 自訂觸發器

用您自己的按鈕或輸入框替換預設的觸發器。`Trigger` 元件接受一個 render prop，讓您可以完全控制觸發器的外觀和行為。

<CodeDemo source={CompoundCustomTriggerSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## 觸發器 Render Props

| Prop           | Type             | 說明                    |
| -------------- | ---------------- | ----------------------- |
| `displayValue` | `string`         | 格式化後的已選日期      |
| `placeholder`  | `string`         | 預留位置文字            |
| `hasValue`     | `boolean`        | 是否已選擇日期          |
| `isOpen`       | `boolean`        | 彈出視窗是否開啟        |
| `onToggle`     | `() => void`     | 切換彈出視窗的開啟/關閉 |
| `onClear`      | `() => void`     | 清除選擇                |
| `locale`       | `Locale`         | 目前的語系物件          |
| `triggerRef`   | `(node) => void` | 用於定位的 Ref 回呼函式 |
