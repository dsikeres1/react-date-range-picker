# 字體排印

styled 套件預設使用**系統字體堆疊**：

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

這會在各個平台上使用作業系統的原生字體進行渲染（macOS 上的 San Francisco，Windows 上的 Segoe UI，Android 上的 Roboto）。系統字體提供了良好的多語言支援，因為作業系統會自動處理中日韓、阿拉伯文和其他腳本的備用字體。

若要覆寫，請在任何祖先元素上設定 `--rdrp-font-family` — 它會串聯至選擇器中。

## 使用網頁字體 (Noto Sans)

對於需要一致跨平台渲染或國際化 (i18n) 的專案，可以載入像 [Noto Sans](https://fonts.google.com/noto) 這樣的網頁字體並覆寫該變數：

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/styles.css";

// 1. 在您的 HTML 或 CSS 中載入 Noto Sans：
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. 覆寫字體變數：
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## 字體大小與字重

透過覆寫大小和字重變數來微調字體排印的密度：

| 變數                          | 預設值 | 描述             |
| ----------------------------- | ------ | ---------------- |
| `--rdrp-font-size-xs`         | `11px` | 最小的文字       |
| `--rdrp-font-size-sm`         | `12px` | 星期標籤         |
| `--rdrp-font-size-base`       | `13px` | 日期儲存格、按鈕 |
| `--rdrp-font-size-md`         | `14px` | 觸發器、時間項目 |
| `--rdrp-font-size-lg`         | `15px` | 月份標題         |
| `--rdrp-font-weight-normal`   | `400`  | 基本字重         |
| `--rdrp-font-weight-medium`   | `500`  | 中等強調         |
| `--rdrp-font-weight-semibold` | `600`  | 已選日期、標題   |
| `--rdrp-font-weight-bold`     | `700`  | 今日指示器       |
