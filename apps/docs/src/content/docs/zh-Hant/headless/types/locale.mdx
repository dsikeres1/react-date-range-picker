# 本地化

所有面向使用者的字串介面。傳遞一個 `Partial<Locale>` 到任何掛鉤 (hook) 以覆寫特定的字串 —— 未指定的欄位將回退到 `DEFAULT_LOCALE`。

```tsx
import type { Locale } from "react-date-range-picker-headless";
```
