# Поведение

## Закрытие при выборе

Когда `shouldCloseOnSelect` включен, выборщик автоматически подтверждает и закрывается при нажатии на предустановку.

<Admonition type="info">
  Для выборщиков с возможностью указания времени, `shouldCloseOnSelect` применяется только к
  нажатиям на предустановки — а не на даты, так как пользователю все еще нужно установить время.
</Admonition>
