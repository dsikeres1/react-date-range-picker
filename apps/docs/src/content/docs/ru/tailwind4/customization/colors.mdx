# Цвета

Переключайте цветовые темы во время выполнения, переопределяя пользовательские свойства CSS для элемента-обертки. Поскольку пакет Tailwind v4 считывает все цвета из семантических токенов, изменение базовых переменных CSS мгновенно меняет оформление всего компонента выбора даты.

Пакет Tailwind v4 использует формат **oklch** для семантических токенов. Переопределите `--color-primary` и `--color-primary-foreground`, чтобы изменить оформление компонента:

  </ExampleWrapper>
</CodeDemo>
