# Пользовательский триггер

Замените стандартную кнопку-триггер своим собственным элементом, используя рендер-проп `Trigger`. Когда вы передаете функцию в качестве дочернего элемента в `DatePicker.Trigger`, вы получаете состояние и колбэки для создания любого желаемого интерфейса триггера.

## Рендер-пропы триггера

| Проп           | Тип              | Описание                                        |
| -------------- | ---------------- | ----------------------------------------------- |
| `displayValue` | `string`         | Отформатированная выбранная дата                |
| `placeholder`  | `string`         | Текст-заполнитель                               |
| `hasValue`     | `boolean`        | Выбрана ли дата                                 |
| `isOpen`       | `boolean`        | Открыто ли всплывающее окно                     |
| `onToggle`     | `() => void`     | Переключить открытие/закрытие всплывающего окна |
| `onClear`      | `() => void`     | Очистить выбор                                  |
| `locale`       | `Locale`         | Текущий объект локали                           |
| `triggerRef`   | `(node) => void` | Ref-колбэк для позиционирования                 |
