# Типографика

Пакет Tailwind v4 использует класс `font-sans` по умолчанию от Tailwind (`ui-sans-serif, system-ui, sans-serif, ...`). Поскольку CSS свойство `font-family` наследуется, вы можете переопределить шрифт компонента, обернув его в контейнер со стилем `fontFamily`. Шрифт из обертки каскадом применится ко всем элементам компонента.

## Семейство шрифтов

Оберните компонент и установите `fontFamily` для обертки, чтобы сменить шрифт:

## Использование веб-шрифтов (Noto Sans)

Загрузите веб-шрифт во время выполнения и примените его с помощью обертки. В этом примере загружается [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) из Google Fonts:

<CodeDemo source={CompoundNotoSansSource}>
  <ExampleWrapper variant="tailwind4" client:visible>
    <CompoundNotoSans client:visible />
  </ExampleWrapper>
</CodeDemo>

<Admonition type="tip" title="Использование в продакшене">
В реальном проекте загружайте шрифт через тег `<link>` в `<head>` вашего HTML или через сборщик CSS, а не внедряйте его во время выполнения. Подход с `useEffect` здесь используется только для демонстрационных целей.
</Admonition>
