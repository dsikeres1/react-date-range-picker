# Ограничения

## Минимальное и максимальное количество дней

Ограничьте длину диапазона с помощью `minDays` и `maxDays` (включительно). Пользователи не могут подтвердить диапазон короче `minDays` или длиннее `maxDays`.

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <MinMaxDays client:visible />
  </ExampleWrapper>
</CodeDemo>

## Недоступные даты

Используйте `isDateUnavailable` для динамического отключения определенных дат. Недоступные даты не могут быть выбраны в качестве начала или конца диапазона.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="tailwind3" client:visible>
    <DisabledDates client:visible />
  </ExampleWrapper>
</CodeDemo>

## Разрешить одну дату

По умолчанию разрешен диапазон, в котором дата начала совпадает с датой окончания. Установите `allowSingleDateInRange={false}`, чтобы требовать выбора как минимум двух разных дат.
