# Поведение

## Закрытие при выборе

Когда `shouldCloseOnSelect` включен, выборщик автоматически подтверждает и закрывается при нажатии на пресет.

<Admonition type="info">
  Для выборщиков с включенным временем `shouldCloseOnSelect` применяется только к нажатиям на
  пресеты, а не к выбору даты, так как пользователю все еще нужно установить время.
</Admonition>
