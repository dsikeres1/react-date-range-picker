# Свойства

| Свойство        | Тип                            | По умолчанию | Описание                                                       |
| --------------- | ------------------------------ | ------------ | -------------------------------------------------------------- |
| `value`         | `Date \| null`                 | --           | Выбранное значение времени (используется только часть времени) |
| `onChange`      | `(date: Date \| null) => void` | --           | Callback при изменении значения                                |
| `time`          | `TimeConfig`                   | --           | Конфигурация панели времени                                    |
| `locale`        | `Partial<Locale>`              | --           | Пользовательская конфигурация локали                           |
| `displayFormat` | `string`                       | --           | Пользовательский формат отображения для триггера               |
| `placeholder`   | `string`                       | --           | Текст-заполнитель для триггера                                 |
| `open`          | `boolean`                      | --           | Управляемое состояние открытия                                 |
| `initialOpen`   | `boolean`                      | `false`      | Начальное состояние открытия (неуправляемое)                   |
| `onOpenChange`  | `(open: boolean) => void`      | --           | Callback при изменении состояния открытия                      |
| `required`      | `boolean`                      | `false`      | Запретить очистку значения                                     |
| `inline`        | `boolean`                      | `false`      | Отображать встроенно без триггера/всплывающего окна            |
| `hideFooter`    | `boolean`                      | --           | Скрыть футер с действиями                                      |
| `name`          | `string`                       | --           | Имя скрытого поля для отправки формы                           |

### TimeConfig

```ts
interface TimeConfig {
  /** Точность отображения времени. По умолчанию "minute". */
  precision?: "hour" | "minute" | "second";
  /** 12-часовой или 24-часовой формат. По умолчанию "24". */
  hourFormat?: "12" | "24";
  /** Шаг приращения минут. По умолчанию 5. */
  minuteStep?: 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
  /** Шаг приращения секунд. По умолчанию 1. */
  secondStep?: 1 | 2 | 3 | 5 | 10 | 15 | 20 | 30;
  /** Высота (в пкс) каждого элемента колеса прокрутки. По умолчанию 32. */
  itemHeight?: number;
}
```
