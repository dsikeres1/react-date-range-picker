# Цвета

Пакет Tailwind v3 использует **жестко закодированные утилитарные классы Tailwind** (например, `bg-sky-500`) вместо переменных CSS. Чтобы изменить цвета, используйте Compound API и динамически переопределяйте имена классов дней:

<Admonition type="info">
  В отличие от пакета styled (который использует переменные CSS), Tailwind v3 требует использования
  Compound API для переопределения цветов во время выполнения. См. [Переопределение
  темы](/docs/tailwind3/customization/theme-override) для получения более подробной информации.
</Admonition>
