# Пользовательский триггер

Замените стандартный триггер вашим собственным элементом, используя render-проп. Компонент `Trigger` принимает дочернюю функцию, которая получает состояние и обратные вызовы, позволяя вам отображать любую пользовательскую кнопку или поле ввода.

## Render-пропсы триггера

| Проп           | Тип              | Описание                                        |
| -------------- | ---------------- | ----------------------------------------------- |
| `displayValue` | `string`         | Отформатированная выбранная дата                |
| `placeholder`  | `string`         | Текст-заполнитель                               |
| `hasValue`     | `boolean`        | Выбрана ли дата                                 |
| `isOpen`       | `boolean`        | Открыто ли всплывающее окно                     |
| `onToggle`     | `() => void`     | Переключить открытие/закрытие всплывающего окна |
| `onClear`      | `() => void`     | Очистить выбор                                  |
| `locale`       | `Locale`         | Текущий объект локали                           |
| `triggerRef`   | `(node) => void` | Callback-ссылка для позиционирования            |
