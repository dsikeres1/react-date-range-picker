# Поведение

## Закрытие при выборе

Когда `shouldCloseOnSelect` включен, календарь автоматически подтверждает и закрывается при нажатии на предустановку.

<Admonition type="info">
  Для календарей с выбором времени, `shouldCloseOnSelect` применяется только к нажатиям на
  предустановки — не к нажатиям на даты, так как пользователю все еще нужно установить время.
</Admonition>
