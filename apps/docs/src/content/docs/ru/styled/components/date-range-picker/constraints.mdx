# Ограничения

## Минимальное и максимальное количество дней

Ограничьте длину диапазона с помощью `minDays` и `maxDays` (включительно). Пользователи не могут подтвердить диапазон короче `minDays` или длиннее `maxDays`.

<CodeDemo source={MinMaxDaysSource}>
  <ExampleWrapper variant="styled" client:load>
    <MinMaxDays client:load />
  </ExampleWrapper>
</CodeDemo>

## Недоступные даты

Используйте `isDateUnavailable` для динамического отключения определенных дат. Недоступные даты не могут быть выбраны в качестве начальной или конечной точки диапазона.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="styled" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## Разрешить одну дату

По умолчанию разрешен диапазон, в котором начальная дата совпадает с конечной. Установите `allowSingleDateInRange={false}`, чтобы требовать как минимум две разные даты.
