# Ограничения

## Обязательное поле

Когда установлено `required`, кнопка очистки отключается, и значение не может быть пустым.

<CodeDemo source={RequiredSource}>
  <ExampleWrapper variant="styled" client:load>
    <Required client:load />
  </ExampleWrapper>
</CodeDemo>

## Минимальная и максимальная даты

Ограничьте выбираемые даты определенным диапазоном с помощью `minDate` и `maxDate`.

<CodeDemo source={MinMaxDateSource}>
  <ExampleWrapper variant="styled" client:load>
    <MinMaxDate client:load />
  </ExampleWrapper>
</CodeDemo>

## Недоступные даты

Используйте `isDateUnavailable` для динамического отключения определенных дат.

<CodeDemo source={DisabledDatesSource}>
  <ExampleWrapper variant="styled" client:load>
    <DisabledDates client:load />
  </ExampleWrapper>
</CodeDemo>

## Отключить прошлое / будущее

Используйте `disablePast` или `disableFuture`, чтобы быстро отключить даты до/после сегодняшнего дня.

<CodeDemo source={DisablePastFutureSource}>
  <ExampleWrapper variant="styled" client:load>
    <DisablePastFuture client:load />
  </ExampleWrapper>
</CodeDemo>

## Выделение дат

Визуально выделяйте определенные даты (например, праздники, события), используя свойство `highlightDates`.

<CodeDemo source={HighlightDatesSource}>
  <ExampleWrapper variant="styled" client:load>
    <HighlightDates client:load />
  </ExampleWrapper>
</CodeDemo>
