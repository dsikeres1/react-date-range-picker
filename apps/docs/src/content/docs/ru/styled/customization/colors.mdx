# Цвета

Переключайте цветовую тему во время выполнения, переопределяя CSS-свойства на элементе-обёртке. Пакет styled использует `--rdrp-color-primary`, `--rdrp-color-primary-hover`, `--rdrp-color-primary-light` и `--rdrp-color-primary-lighter` для всех основных акцентных цветов.

## Встроенный переключатель тем

<CodeDemo source={CompoundColorThemeSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorTheme client:visible />
  </ExampleWrapper>
</CodeDemo>

## Переключатель тем во всплывающем окне

Тот же подход работает и для выбора дат во всплывающем окне:

<CodeDemo source={CompoundColorThemePopupSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CompoundColorThemePopup client:visible />
  </ExampleWrapper>
</CodeDemo>

## Встроенный стиль с CSS-переменными

Вы также можете переопределять цвета напрямую через атрибут `style` на элементе-обёртке:

<CodeDemo source={CSSCustomColorsSource}>
  <ExampleWrapper variant="styled" client:visible>
    <CSSCustomColors client:visible />
  </ExampleWrapper>
</CodeDemo>
