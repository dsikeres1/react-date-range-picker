# Пользовательский триггер

Замените стандартный триггер своей собственной кнопкой или полем ввода. Компонент `Trigger` принимает `render prop`, который дает вам полный контроль над внешним видом и поведением триггера.

  <ExampleWrapper variant="styled" client:visible>
    <CompoundCustomTrigger client:visible />
  </ExampleWrapper>
</CodeDemo>

## Render Props для триггера

| Проп           | Тип              | Описание                           |
| :------------- | :--------------- | :--------------------------------- |
| `displayValue` | `string`         | Отформатированная выбранная дата   |
| `placeholder`  | `string`         | Текст-заполнитель                  |
| `hasValue`     | `boolean`        | Выбрана ли дата                    |
| `isOpen`       | `boolean`        | Открыто ли всплывающее окно        |
| `onToggle`     | `() => void`     | Переключить открытие/закрытие окна |
| `onClear`      | `() => void`     | Очистить выбор                     |
| `locale`       | `Locale`         | Текущий объект локали              |
| `triggerRef`   | `(node) => void` | Ref-колбэк для позиционирования    |
