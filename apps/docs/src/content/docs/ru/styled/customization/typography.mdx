# Типографика

Пакет `styled` по умолчанию использует **системный стек шрифтов**:

```css
--rdrp-font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

Это обеспечивает рендеринг с использованием нативного шрифта ОС на каждой платформе (San Francisco на macOS, Segoe UI на Windows, Roboto на Android). Системные шрифты предоставляют хорошую поддержку нескольких языков, поскольку ОС автоматически обрабатывает резервные варианты для CJK, арабского и других письменностей.

Чтобы переопределить, установите `--rdrp-font-family` на любом родительском элементе — он каскадно применится к компоненту выбора даты.

## Использование веб-шрифтов (Noto Sans)

Для согласованного кросс-платформенного рендеринга или проектов с интернационализацией (i18n) загрузите веб-шрифт, такой как [Noto Sans](https://fonts.google.com/noto), и переопределите переменную:

```tsx
import { DatePicker } from "react-date-range-picker-styled";
import "react-date-range-picker-styled/rdrp-styles.css";

// 1. Load Noto Sans in your HTML or CSS:
//    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">

// 2. Override the font variable:
<div style={{ "--rdrp-font-family": '"Noto Sans", sans-serif' } as React.CSSProperties}>
  <DatePicker value={value} onChange={setValue} inline />
</div>;
```

## Размер и насыщенность шрифта

Точно настройте плотность типографики, переопределяя переменные размера и насыщенности:

| Переменная                    | По умолчанию | Описание                   |
| ----------------------------- | ------------ | -------------------------- |
| `--rdrp-font-size-xs`         | `11px`       | Самый маленький текст      |
| `--rdrp-font-size-sm`         | `12px`       | Метки дней недели          |
| `--rdrp-font-size-base`       | `13px`       | Ячейки дней, кнопки        |
| `--rdrp-font-size-md`         | `14px`       | Триггер, элементы времени  |
| `--rdrp-font-size-lg`         | `15px`       | Заголовок месяца           |
| `--rdrp-font-weight-normal`   | `400`        | Базовая насыщенность       |
| `--rdrp-font-weight-medium`   | `500`        | Средний акцент             |
| `--rdrp-font-weight-semibold` | `600`        | Выбранные дни, заголовки   |
| `--rdrp-font-weight-bold`     | `700`        | Индикатор сегодняшнего дня |
